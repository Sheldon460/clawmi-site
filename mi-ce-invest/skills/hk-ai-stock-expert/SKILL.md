# 港股 AI 投研专家技能 (HK AI Stock Expert)

## 核心职能

作为幂家军的行情分析师 (mi-ce-invest)，本技能专注**港股 AI/科技板块深度研究**，提供南向资金分析、产业链研究和投资建议。

## 当以下情况时使用此技能

✅ **USE this skill when:**

- "分析港股 AI 板块投资机会"
- "腾讯/阿里/美团深度研究"
- "南向资金流向分析"
- "港股科技股估值对比"
- "AI 产业链上下游分析"
- "港股通持仓变化"
- "大行评级汇总"
- "港股 IPO 分析"

❌ **DON'T use this skill when:**

- A 股分析 → 使用 `stock-analysis` 技能
- 实时行情查询 → 使用 `stock-monitor` 技能
- 美股分析 → 使用 `stock-analysis` (支持美股)
- 衍生品分析 → 使用期权/期货工具

## 分析方法

### 1. 南向资金分析

```python
# 核心指标
- 南向资金净流入/流出
- 港股通持仓变化
- 连续 N 日净买入/卖出
- 占流通股比例变化

# 数据源
- 港交所披露易
- 东方财富Choice
- 沪深交易所港股通数据
```

### 2. 基本面分析 (港股特色)

```python
# 财务指标
- 营收 (分业务拆分)
- 经调整净利润 (Non-GAAP)
- 自由现金流
- 用户数据 (MAU/ARPU)

# 估值指标
- PE (动态/静态)
- PS (市销率，适用未盈利公司)
- PEG
- EV/EBITDA
```

### 3. 产业链分析

```python
# AI 产业链图谱
- 算力层：芯片、服务器、数据中心
- 算法层：大模型、框架、工具链
- 应用层：ToB (企业)、ToC (消费者)

# 竞争格局
- 市场份额
- 技术壁垒
- 生态优势
```

### 4. 大行评级汇总

```python
# 投行评级
- 高盛、摩根士丹利、中金、中信
- 目标价汇总
- 评级变化追踪
- 研报观点提炼
```

## 使用方法

### 1. 个股深度分析

```bash
# 分析单只港股
python3 hk_ai_expert.py --symbol 00700.HK --report full

# 分析 AI 板块
python3 hk_ai_expert.py --sector AI --report industry

# 南向资金分析
python3 hk_ai_expert.py --symbol 00700.HK --southbound-flow
```

### 2. 产业链研究

```bash
# AI 产业链分析
python3 hk_ai_expert.py --chain AI --depth deep

# 竞争格局分析
python3 hk_ai_expert.py --compare 00700.HK,09988.HK,03690.HK

# 估值对比
python3 hk_ai_expert.py --valuation-compare tech-sector
```

### 3. 资金流向监控

```bash
# 南向资金日报
python3 hk_ai_expert.py --southbound-daily

# 港股通持仓变化
python3 hk_ai_expert.py --holdings-change --period weekly

# 大行评级汇总
python3 hk_ai_expert.py --analyst-ratings --symbol 00700.HK
```

## 数据源

| 数据类型 | 数据源 | 延迟 |
|----------|--------|------|
| 实时行情 | 新浪财经 | 15 分钟 |
| 南向资金 | 港交所/东方财富 | T+1 |
| 财务数据 | 披露易/Choice | 季报披露后 |
| 大行评级 | 投行研报 | 实时 |
| 持仓变化 | 港交所披露易 | T+1 |

## 注意事项

1. **数据延迟**: 港股行情有 15 分钟延迟
2. **汇率风险**: 港币挂钩美元，注意汇率波动
3. **流动性**: 部分小市值股票流动性较差
4. **政策风险**: 关注港股通政策变化
