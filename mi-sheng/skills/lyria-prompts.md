# 幂声 Lyria 音乐生成 Prompt 模板库 V1.0

**用途**: 用于 Google Lyria (lyria-002 模型) 生成背景音乐  
**协作对象**: mi-wen (文案)、mi-ying (视频)、mi-hua (视觉)

---

## 📋 Prompt 结构公式

```
[情绪氛围] + [乐器配置] + [节奏速度] + [时长要求] + [使用场景] + [参考风格]
```

---

## 🎭 情绪氛围库

### 欢快积极
- 欢快、阳光、充满活力、令人振奋、轻松愉快
- 适用：开场、转场、产品展示、成就时刻

### 温馨感人
- 温暖、治愈、感人、柔情、怀旧
- 适用：故事叙述、情感共鸣、回忆片段

### 紧张悬疑
- 紧张、悬疑、压迫感、神秘、危机四伏
- 适用：剧情高潮、悬念铺垫、惊悚场景

### 专业商务
- 专业、稳重、可靠、科技感、现代
- 适用：企业宣传、产品介绍、数据展示

### 史诗宏大
- 史诗、宏大、震撼、壮丽、辉煌
- 适用：片头、预告片、重大时刻

## ⏱️ 节奏速度库

| 术语 | BPM | 适用场景 |
|------|-----|---------|
| 慢板 (Adagio) | 60-80 | 情感叙述、悲伤场景 |
| 中板 (Moderato) | 80-120 | 日常内容、对话背景 |
| 快板 (Allegro) | 120-160 | 活力展示、运动场景 |
| 急板 (Presto) | 160+ | 高潮片段、紧张追逐 |

## 📝 完整 Prompt 模板

### 模板 01: 企业宣传片
```
专业且现代的背景音乐，以钢琴和弦乐为主，中板节奏 (100BPM)，时长 60 秒。
营造可靠、科技感、创新的氛围，适合企业宣传片开场。
参考风格：Apple 产品发布会背景音乐。
```

### 模板 02: 情感故事片
```
温暖治愈的钢琴独奏，轻柔的弦乐衬托，慢板节奏 (70BPM)，时长 90 秒。
营造怀旧、感人、回忆的氛围，适合个人故事叙述。
参考风格：电影《寻梦环游记》配乐风格。
```

### 模板 03: 产品展示视频
```
欢快活力的电子音乐，合成器脉冲贝斯，快板节奏 (128BPM)，时长 45 秒。
营造现代、时尚、充满能量的氛围，适合科技产品展示。
参考风格：TED 演讲开场音乐。
```

### 模板 04: 纪录片旁白背景
```
简约的环境音乐，轻柔的钢琴和氛围音效，中慢板 (80BPM)，时长 120 秒。
营造沉思、探索、知识的氛围，适合纪录片旁白背景。
音量需保持在旁白之下，不喧宾夺主。
```

### 模板 05: 预告片/片头
```
史诗宏大的管弦乐团配乐，渐进式结构，从安静到爆发，时长 60 秒。
包含定音鼓、铜管乐器、合唱团，营造震撼、壮丽的氛围。
适合电影预告片或系列节目片头。
```

### 模板 06: 轻松日常 Vlog
```
轻松愉快的原声吉他，温暖的指弹风格，中板节奏 (100BPM)，时长 180 秒。
营造日常、亲切、放松的氛围，适合 Vlog 日常生活记录。
可循环播放，无缝衔接。
```

### 模板 07: 悬疑剧情
```
紧张悬疑的氛围音乐，低音弦乐持续音，不规则节奏，时长 90 秒。
营造神秘、压迫感、危机四伏的氛围，适合悬疑剧情铺垫。
使用不和谐音程和突然的音效转折。
```

### 模板 08: 中国风内容
```
中国传统乐器 (古筝、笛子、二胡) 与现代节奏融合，中板 (90BPM)，时长 75 秒。
营造古典、优雅、东方美学的氛围，适合传统文化内容。
参考风格：故宫文创宣传片配乐。
```

## 🎯 Lyria 调用命令示例

```bash
# 基础调用
lyria generate \
  --prompt "温暖治愈的钢琴独奏，轻柔弦乐衬托，慢板 70BPM，时长 90 秒" \
  --model lyria-002 \
  --output output.mp3

# 带参数调用
lyria generate \
  --prompt "专业现代的企业宣传片音乐，钢琴 + 弦乐，中板 100BPM" \
  --model lyria-002 \
  --duration 60 \
  --format mp3 \
  --bitrate 320 \
  --output company_intro_bgm.mp3
```

## 🔄 与 mi-ying 协作流程

1. **需求接收**: mi-ying 提供视频场景描述
2. **Prompt 选择**: 从模板库选择或定制 Prompt
3. **音乐生成**: 使用 Lyria 生成 2-3 个版本供选择
4. **反馈调整**: 根据 mi-ying 反馈调整 Prompt
5. **最终交付**: 输出 WAV 格式 (无损) + MP3 格式 (预览)

## 📊 质量检查清单

- [ ] 音乐情绪与视频场景匹配
- [ ] 时长符合要求 (或可无缝循环)
- [ ] 动态范围适中 (不过于激烈)
- [ ] 频率分布合理 (不掩盖人声)
- [ ] 无突兀的转折或噪音
- [ ] 格式正确 (WAV/MP3)
- [ ] 元数据完整 (标题、作者、版权)
