# 项目追踪系统

## 📁 项目文件夹结构
```
projects/
├── 项目 A/
│   ├── info.md          # 项目基本信息
│   ├── timeline.md      # 时间线
│   └── logs/            # 相关日志
├── 项目 B/
└── _index.md            # 所有项目索引
```

## 📊 项目状态
- 🟢 进行中
- 🟡 待启动
- 🔵 暂停
- ✅ 已完成
- ❌ 已取消
