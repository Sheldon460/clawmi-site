## 项目目标
建立一个完整的日常记忆系统，实现：
1. 每天工作内容自动日志记录
2. 讨论结束后自动写摘要存入追踪系统
3. 支持按日期/项目/关键词查询
4. 提供 Dashboard 可视化浏览

## 系统架构
```
日常记忆系统
├── 日志记录 (daily-logs/)
├── 摘要生成 (summaries/)
├── 项目追踪 (projects/)
├── Dashboard (dashboard/)
└── 自动化脚本 (scripts/)
```
