# 日常记忆系统 - 每日日志

## 📁 文件命名规范
- 格式：`YYYY-MM-DD.md`
- 示例：`2026-03-25.md`

## 📝 日志结构

```markdown
# 工作日志 - YYYY-MM-DD

## 📊 今日概览
- **日期**: YYYY-MM-DD
- **星期**: X
- **工作时长**: X 小时
- **讨论次数**: X 次
- **完成项目**: X 个

## 💬 讨论记录

### 讨论 #1 - [主题]
- **时间**: HH:MM
- **参与人**: XXX
- **内容摘要**: ...
- **关键决策**: ...
- **待办事项**: ...

### 讨论 #2 - [主题]
...

## 📌 项目进展

| 项目 | 进展 | 状态 |
|------|------|------|
| 项目 A | 完成 XX | 🟢 |
| 项目 B | 进行中 | 🟡 |

## 📝 备注
...
```

## 🔧 自动化
- 每日 23:00 自动生成日志框架
- 讨论结束后自动追加记录
- 每周日生成周报
