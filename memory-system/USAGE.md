# 日常记忆系统 - 使用指南

## 🎯 系统功能

1. **自动日志记录** - 每天工作内容自动记录
2. **讨论摘要生成** - 每次讨论后自动生成摘要
3. **项目追踪** - 按项目分类追踪进展
4. **智能检索** - 按日期/项目/关键词查询
5. **Dashboard** - 可视化浏览所有记录

## 🤖 自动化流程

### 1. 每日日志自动创建
- **触发时间**: 每日 00:00
- **动作**: 自动创建当日日志文件
- **位置**: `daily-logs/YYYY-MM-DD.md`

### 2. 讨论摘要自动生成
- **触发条件**: 用户说"结束讨论"或"生成摘要"
- **动作**: 自动总结讨论内容并保存
- **位置**: `summaries/YYYYMMDD-HHMM-XXX-主题.md`

### 3. 待办事项自动追踪
- **触发条件**: 讨论中识别到待办任务
- **动作**: 自动添加到待办列表
- **同步**: 更新项目追踪表

## 📝 使用命令

### 创建日志
```bash
# 手动创建今日日志
./scripts/create-daily-log.sh
```

### 生成摘要
```bash
# 生成讨论摘要
./scripts/generate-summary.sh 20260325-1435-001
```

### 查询记录
```bash
# 查看某日日志
cat daily-logs/2026-03-25.md

# 搜索关键词
grep -r "模型切换" summaries/

# 查看项目进展
cat projects/项目名/info.md
```
