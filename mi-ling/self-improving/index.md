# Self-Improving 索引 - 幂领 (mi-ling)

## 文件结构

```
self-improving/
├── index.md          # 本文件 - 索引
├── memory.md         # 个人技能记忆
└── corrections.md    # 错误记录
```

## 快速导航

### 记忆类型
- **Patterns**: 有效的工作模式
- **Preferences**: 用户偏好
- **Rules**: 特定规则

### 错误类型
- **API Errors**: API调用错误
- **Logic Errors**: 逻辑错误
- **Output Errors**: 输出格式错误

## 使用说明

1. **任务启动时**: 读取 memory.md 和 corrections.md
2. **发现错误时**: 立即写入 corrections.md
3. **任务结束时**: 如有改进点，写入 memory.md
4. **声明格式**: `[自我进化] 已记录到: ...`
