## 🚀 快速部署

### 本地预览
```bash
cd /tmp/clawd_bridge/data/mi-army/mi-ling/clawmi-site
./preview.sh
```

### 部署到 Vercel
```bash
vercel --prod
```

### 部署到 Cloudflare Pages
```bash
wrangler pages deploy dist --project-name=clawmi
```
