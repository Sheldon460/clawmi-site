# SBTI 人格测试网站 - Stitch UI 设计生成 Prompt

**产品名 + 定位**: SBTI — MBTI过时了，试试这个荒诞幽默的31道戏仿人格测试，测出你的隐藏人格

**目标用户**: 95后、00后、10后年轻用户、喜欢玩梗的社交媒体用户、MBTI 爱好者

**配色要求**: 淡绿白底（#f6faf6）+ 清新绿主色（#6c8d71）+ 深绿强调色（#4d6a53）+ 浅绿辅助色（#dbe8dd）+ 深绿黑文字（#1e2a22）

**页面结构**:
1. Hero Screen: 单列居中布局，大标题"MBTI已经过时，SBTI来了"，居中"开始测试"按钮，淡绿径向渐变背景（radial-gradient circle at top left）
2. Test Screen: 单列布局，顶部绿色渐变进度条 + 进度文字（0/31），问题卡片堆叠（每题4个选项，单选样式），底部操作按钮栏（返回首页、提交查看结果）
3. Result Screen: 左右布局（桌面端）/单列布局（移动端），左侧海报图（大字人格代码），右侧结果信息（类型名称 + 匹配度 + 解读），维度评分列表，作者折叠卡片，底部操作按钮（重新测试、回到首页）
4. Footer: 简洁版权信息，"仅供娱乐"提示

**消息层级**:
- 主标题: MBTI已经过时，SBTI来了
- 副标题: 31道荒诞幽默的戏仿人格测试，测出你的隐藏人格
- CTA: 开始测试

**定价信息**: 免费（无需定价）

**设计调性**: Playful、Gen Z、Minimalist、Humor、Light、Fresh

**反 AI 味要求**:
- 不要紫蓝渐变背景，用淡绿径向渐变
- 不要对称 3 列布局，用单列居中或左文右图布局
- 不要用 emoji 做 icon，用极简线条图标或无图标
- 不要每个 section 背景色交替白/灰，保持统一淡绿背景
- 不要圆角卡片堆砌，用大圆角（rounded-3xl）但适度间距
- 不要默认 placeholder 头像，用人格代码大字作为视觉元素
- 不要过度使用阴影，用轻盈的半透明阴影（rgba(47, 73, 55, 0.08)）
- 不要复杂的 3D 动画，用简单的悬停缩放（hover:-translate-y-1）

---

## 技术要求
- 使用 Tailwind CSS
- 支持 Next.js 14
- 响应式设计（移动端友好）
- 保持原有逻辑不变

## 需要生成的屏幕
1. Hero Screen - 首页入口
2. Test Screen - 测试问题页
3. Result Screen - 结果展示页
