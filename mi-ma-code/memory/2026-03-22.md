## 技术笔记

### Next.js 16 部署配置变更
Next.js 16 默认启用 Turbopack，需要：
1. 移除 `next.config.ts` 中的 `webpack` 配置
2. 添加 `turbopack: {}` 配置（可以是空对象）
3. 静态导出时 `outputDirectory` 应设为 `.next`

### Vercel 部署模式
- **云端构建**: `vercel --prod` - 在 Vercel 服务器构建
- **本地构建+部署**: `vercel build --prod && vercel deploy --prebuilt --prod` - 本地构建后上传
