## ⚠️ 遇到的问题

### 问题 1：web_search 需要 API 密钥
**错误信息**：`missing_perplexity_api_key`
**解决方案**：
- 需要在 OpenClaw 配置中设置 `PERPLEXITY_API_KEY`
- 或者使用飞书定时机器人方案，通过 mi-zhi 的 web_search 工具调用

### 问题 2：launchd 加载失败
**错误信息**：`Bootstrap failed: 5: Input/output error`
**原因**：路径中包含空格（`/Volumes/My house/`）
**解决方案**：
- 改用 cron 方案
- 或者将文件移动到无空格路径

### 问题 3：网络访问限制
**错误信息**：`Blocked: resolves to private/Internal/special-use IP address`
**原因**：web_fetch 工具的安全限制
**解决方案**：
- 使用 web_search（需要 API 密钥）
- 或者通过 OpenClaw 内置工具代理
