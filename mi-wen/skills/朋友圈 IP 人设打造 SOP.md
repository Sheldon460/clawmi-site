# 朋友圈 IP 人设打造 SOP (V1.0)

> 5 类朋友圈文案打造高价值吸粉 IP 人设

## 🎯 核心目标

通过 5 类朋友圈文案，打造立体、可信、高价值的 IP 人设：
1. **生活类**：真实可信的活人感
2. **好评反馈类**：有结果、被信任的专家人设
3. **专业干货类**：高价值的实力人设
4. **产品推广类**：不惹人厌的创收人设
5. **互动类**：高粘性的朋友人设

---

## 一、生活类文案：打造真实可信的活人感

### 💡 核心理念
**作用**：向用户展示一个立体、真实、有血有肉的 IP，建立基础信任。

### 📝 写作要点

#### 1. 写什么？
**误区**：一定要等到有重大事件才发朋友圈。

**正确做法**：生活中各种小细节都可挖掘。

**可写内容**：
- 起床、买菜、做饭、运动
- 读书、思考、与他人相处
- 生活中的每一件事

#### 2. 怎么写？
**最大原则**：**感知细节**

### 🎯 案例对比

#### ❌ 普通版
> 起床啦，一天之计在于晨，新的一天，加油！

#### ✅ 细节版（版本 1）
> 它轻轻抚摸我的脸庞，让我逐渐感受到它的温度，仿佛告诉我：该起床了，再不起迟到啦！它是谁？哈哈，大太阳！

#### ✅ 细节版（版本 2）
> 臭肥猫叫我起床的方式终于不再是压在身上把我沉醒，它改了，改舔脸了！准备明天教它给我化妆！

#### ✅ 细节版（版本 3）
> 清晨起床，叫醒我的是梦想吗？不，是我妈在门外的叮叮咣咣！

> 💡 **总结**：普通文案因为加了细节，就变得更加温馨更有趣！场景感很强。

## 二、好评反馈类文案：打造有结果、被信任的专家人设

### 💡 核心理念
**不仅仅局限在"用了之后的好处变化"**。

从有人咨询到最终成交，从购买者开始使用到最终的变化影响，**整个过程都可能藏着好评反馈**。

### 📝 写作要点

#### 1. 素材来源
- 客户的好评、报喜
- 用户购买前的疑虑 → 使用后的反馈（态度对比）
- 群里的精彩片段
- 解决意向用户顾虑的内容

#### 2. 写作技巧
**善于捕捉细节**，把一个很小的故事讲述得曲折有趣。

### 🎯 参考公式
```
开头：用户特征
正文：故事的起因经过
结尾：进行相关解释及宣传
```

### 📌 案例示例

#### 案例 1：哺乳期宝妈咨询
> 哺乳期宝妈来咨询××产品……
> 
> 经过是这样的……
> 
> 请大家放心，哺乳期也可以用！
> 我们好多哺乳期妈妈用了，都反馈……

#### 案例 2：19 岁大学生咨询
> 19 岁的大一学生就这么有思考力了吗？……
> 
> 今天有一个在读大一的小伙伴儿找我聊天……
> 
> 对于××星球，有这些顾虑太正常啦！
> 我们也有个学员 XXX……

#### 案例 3：学员赚钱报喜
> 别怀疑，xx 助理培训班真的可以赚到钱
> 
> 最近好多小伙伴找我报喜，说他们上岗啦赚钱啦，看我家学员们怎么说……
> 
> 还在观望的小伙伴，欢迎咨询！

> 💡 **重点提示**：【重点前置】原则！在朋友圈文案的开头，就把整个文案的重点写出来。

## 三、专业干货类文案：打造高价值的实力人设

### 💡 核心理念
**IP 品牌 = 你 + 你的专业度**

你越值得信任，在某一个领域越专业，那你的个人品牌就会树立得越突出、越坚实。

### 📝 写作要点

#### 1. 内容形式
**干货形式**呈现：
- 一个他们不知道的知识点
- 一个能提高效率的方法
- 一个独特的思考视角
- 一个对行业趋势的洞察

#### 2. 写作方法
**忘掉生硬的自夸，学会分享有价值的信息。**

### 🎯 参考公式
```
开头：提出一个具体问题/痛点/好奇点（戳中用户）
正文：分享你的方法论、技巧或独特见解（提供价值）
结尾：引导思考或轻松互动（深化印象）
```

### 📌 案例对比

#### ❌ 旧思路（秀成绩）
> 本月公众号收入 XXX 元！

#### ✅ 新思路（秀干货）
> 常被学员问到，公众号靠什么赚钱？分享一张我整理的变现渠道图【配图】，主流玩法都在里面了。
> 
> 其实核心就一点"提供价值"。你提供的价值越稀缺，变现路径就越短。你正在做哪个渠道？

#### ❌ 旧思路（秀成绩）
> 一篇文章涨粉 XXXX 个！

#### ✅ 新思路（秀干货）
> 写好文章结尾，能让涨粉效率翻倍！分享 3 个我百试不爽的'钩子型'结尾模板：
> 
> 1. **提问式**：……
> 2. **福利式**：……
> 3. **共鸣式**：……
> 
> 你最喜欢用哪一种？【配文章截图案例】

## 四、产品推广类文案：打造不惹人厌的创收人设

### 💡 核心理念
**不要硬广，要软植入。**

### 📝 写作要点

#### 1. 避免硬广
- 不要直接说"快来买"
- 不要过度吹嘘产品
- 不要频繁发布广告

#### 2. 软植入技巧
- 分享使用场景
- 讲述用户故事
- 展示使用效果
- 提供限时福利

### 🎯 参考公式
```
开头：场景/痛点引入
正文：产品如何解决 + 用户反馈
结尾：限时福利/行动号召
```

## 五、互动类文案：打造高粘性的朋友人设

### 💡 核心理念
**让用户参与进来，而不是单向输出。**

### 📝 写作要点

#### 1. 互动形式
- 提问式：让用户回答
- 投票式：让用户选择
- 征集式：让用户分享
- 福利式：让用户参与

#### 2. 互动技巧
- 问题要简单易答
- 话题要有共鸣
- 及时回复评论
- 精选优质留言

### 🎯 案例示例

#### 提问式
> 大家周末都怎么过？
> A. 宅家追剧
> B. 出门逛街
> C. 学习充电
> D. 其他（评论区告诉我）

#### 征集式
> 一人一句，你听过最扎心的话是什么？
> 我先来："你很好，但是我们不合适。"

## 六、朋友圈发布节奏建议

### 📅 发布频率
- **新手期**：每天 1-2 条（培养习惯）
- **成长期**：每天 3-5 条（增加曝光）
- **成熟期**：每天 5-8 条（矩阵运营）

### ⏰ 发布时间
| 时段 | 适合内容 | 用户状态 |
|------|----------|----------|
| 7:00-9:00 | 生活类、正能量 | 通勤路上 |
| 12:00-13:00 | 干货类、互动类 | 午休时间 |
| 18:00-20:00 | 产品类、好评类 | 下班放松 |
| 21:00-23:00 | 生活类、情感类 | 睡前刷手机 |

### 📊 内容配比建议
| 类型 | 占比 | 说明 |
|------|------|------|
| 生活类 | 40% | 建立信任基础 |
| 好评反馈类 | 20% | 展示实力 |
| 专业干货类 | 20% | 提供价值 |
| 产品推广类 | 10% | 创收变现 |
| 互动类 | 10% | 增加粘性 |

## 七、AI 辅助写作提示词

### 生活类文案提示词
```
请你作为一名朋友圈文案高手，帮我写一条生活类朋友圈文案。

要求：
1. 主题：[例如：清晨起床/做饭/运动/读书]
2. 加入丰富的细节描写（视觉、听觉、嗅觉、触觉）
3. 语言风格：[幽默/温馨/励志/吐槽]
4. 字数：50-150 字
5. 避免 AI 味，要像真人随手发的

示例参考：
"臭肥猫叫我起床的方式终于不再是压在身上把我沉醒，它改了，改舔脸了！准备明天教它给我化妆！"
```

### 好评反馈类文案提示词
```
请你作为一名朋友圈文案高手，帮我写一条好评反馈类朋友圈文案。

要求：
1. 用户特征：[例如：哺乳期宝妈/大学生/职场新人]
2. 故事结构：用户咨询 → 疑虑 → 使用 → 反馈
3. 加入细节描写（对话、心理活动）
4. 结尾自然植入产品/服务
5. 字数：100-200 字
6. 重点前置，开头就点明核心信息
```

### 专业干货类文案提示词
```
请你作为一名朋友圈文案高手，帮我写一条专业干货类朋友圈文案。

要求：
1. 主题：[例如：公众号变现/时间管理/职场沟通]
2. 开头：提出一个痛点/好奇点
3. 正文：分享 3 个实用技巧/方法
4. 结尾：引导互动（提问/讨论）
5. 字数：150-300 字
6. 避免说教，用分享的语气
```

## 八、检查清单

发布前检查：
- [ ] 是否有细节描写（非泛泛而谈）

- [ ] 是否有价值（用户能得到什么）
- [ ] 是否有互动点（引导评论）
- [ ] 是否符合人设（与 IP 定位一致）
- [ ] 配图是否合适（清晰、相关）
- [ ] 发布时间是否合适（用户活跃时段）
