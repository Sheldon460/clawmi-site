# IP类朋友圈写作 SOP (V1.0)

> 核心原则：不要晒生活，要表达生活
> **Express Life, Not Just Show It**

---

## 🎯 核心理念

### 晒 vs 表达的本质区别

| 维度 | 晒生活 | 表达生活 |
|------|--------|----------|
| **句式** | 陈述句："今天我吃了日料" | 感叹句/疑问句："这家日料让我想起京都的那场雨" |
| **内容** | 记录流水账 | 有情感、有故事、有观点 |
| **视角** | "我的" - 只是自己经历的 | "我们的" - 能引发读者共鸣 |
| **效果** | 独立的碎片 | 连续的剧集 |
| **目的** | 追求点赞 | 建立连接 |
| **价值** | 记录瞬间 | 传递价值 |

### 对比案例

#### A组（晒 - 陈述句）
```
今天的早餐，颜值在线。
周末来逛798，人好多。
新买的书到了，开读。
```

#### B组（表达 - 感叹句 + 思考 + 价值）
```
早餐做了开放式三明治，孩子说像艺术品舍不得吃。我想，美育不就是这么一点点渗透进日常的吗？

798新展，人多到挤不进去。索性在街边咖啡馆坐了一下午，看人来人往。有时候，错过展览，反而遇见了生活。

收到一本书，是五年前读过的那本。翻开来，当年的划线还在，但旁边批注的想法，已经完全不一样了。人真是会成长的，连读同一本书的方式都会变。
```

**差异解析**：
- A组：发生了什么（事实）
- B组：发生了什么 + 我想到了什么 + 这对你有什么用（价值）

## 🚀 从"晒"到"表达"的进化路径

### 第一步：加一点感受（扩句法）

使用**五感法**加入自己的感受与感悟：

- **看见了什么？**（视觉细节）
- **听到了什么？**（声音、对话）
- **闻到了什么？**（气味、味道）
- **触动了什么？**（情绪、回忆）
- **想到了什么？**（感悟、观点）

**示例**：

```
晒：今天的咖啡真好喝。
表达：这杯手冲咖啡的香气，让我想起研究生时代泡图书馆的日子。那时候觉得苦，现在回看，却是人生最纯粹的时光。
```

### 第二步：加一点价值（4类价值）

问自己：**这段经历，能给别人带来什么？**

| 价值类型 | 定义 | 判断标准 |
|----------|------|----------|
| **情绪价值** | 让人开心、感动、治愈 | 能让人会心一笑吗？ |
| **干货价值**' | 提供方法、技巧、知识 | 能让人学到东西吗？ |
| **情感价值** | 引发共鸣、认同感 | 能让人产生"我也是这样"的感觉吗？ |
| **向往价值** | 让人想成为你这样的人 | 能让人想拥有你的生活吗？ |

**只要占了其中一条，这条内容就对你的个人IP有了价值。**

## 📝 IP类朋友圈写作模板

### 模板1：生活感悟类（强调成长/变化）

**结构**：场景细节 + 当时的感受 + 现在的感悟 + 给读者的启发

```
[场景描述]
[具体的细节感受]

[转折/对比]
[现在的想法/变化]

[价值升华]
[给读者的一句话启发]
```

**示例**：
```
今天整理书房，翻出了十年前的日记本。

翻到2014年那页，写满了我对未来的焦虑：怕找不到好工作，怕比不上同龄人，怕30岁还一事无成。

现在看当时的自己，真想拍拍他的肩膀说：别担心，你后来的路，比想象的好走多了。

其实我们就是这样走过来的——回头看时，那些让你睡不着觉的焦虑，最后都成了垫脚石。

如果有时光机，你最想对十年前的自己说什么？
```

### 模板2：产品/生活方式类（晒不如表达）

**结构**：物品/行为 + 背后故事 + 它给你的价值 + 读者可以借鉴的地方

```
[物品/行为介绍]

[为什么选择它？/背景故事]
[具体经历/场景]

[它给你的价值/改变]
[读者可以如何应用]
```

**示例**：
```
新入手的这个时间管理本，已经用了三个月。

起因是去年总是被deadline追着跑，累到崩溃。后来发现，不是事情多，是我不知道哪些事情值得做。

这个本子让我养成了一个习惯：每天早上写下"今天最重要的3件事"，晚上复盘。

三个月后，加班少了，产出反而高了。

如果你也觉得"忙但没成果"，不妨试试这个方法：每天只做3件事，但把这3件事做到极致。
```

### 模板3：事件/经历类（从记录到表达）

**结构**：发生了什么 + 我想到了什么 + 这对你有什么用

```
[事件描述]
[时间/地点/人物]

[我的感受/思考]
[当时的情绪/观察]

[对读者的价值]
[情绪价值/干货价值/情感价值/向往价值]
```

**示例**：
```
周六带孩子去自然博物馆，比我想象的挤。

但看到孩子趴在玻璃前看恐龙化石的眼神，我突然明白：重要的不是去了哪里，而是和谁一起。

我小时候，父母忙着赚钱，很少陪我出去玩。所以我决定，无论多忙，每个月至少有一天，完全属于孩子。

不是要带他见多少世面，而是要让他记住：爸爸妈妈愿意花时间，陪我做我喜欢的事。

你小时候有什么记忆，是父母给你的？
```

## ⚡ IP类朋友圈写作心法

### 1. 生活是表达，不是展示

- ❌ 晒产品、晒打卡、晒消费
- ✅ 晒生活方式、晒价值观、晒成长轨迹

### 2. 你的朋友圈是一部连续剧

- 每一条都是连续剧中的一集
- 有自己的主线（成长、价值观、生活方式）
- 有自己的人设（靠谱、有趣、有思考）

### 3. 你呈现的不是生活，是活法

- 输出的不是内容，是样本
- 当你的生活开始给别人"向往感"或借鉴意义
- 人们会主动靠近你，想成为你

### 4. 从今天起改变

- 把"今天我去了哪" → 变成 → "今天我感受到了什么"
- 把"我买了什么" → 变成 → "这件事教会了我什么"

## 🎨 IP类朋友圈的终极形态

**当你的个人IP成为一种"人生样本"**

- 你不需要推销任何产品
- 人们会主动想成为你
- 想拥有你的生活
- 想购买你推荐的一切

**因为真正动人的，从来不是你过得有多好，而是你如何过好每一天，又如何把这些好，变成照亮别人的光。**

## 📊 与5类文案的融合

在原有的5类文案基础上，IP类朋友圈的"表达"原则可以应用于所有类型：

| 文案类型 | 传统写法 | IP表达写法 |
|----------|----------|------------|
| 生活类 | 记录日常 | 日常 + 感受 + 思考 |
| 好评反馈类 | 展示好评 | 好评 + 故事 + 价值 |
| 专业干货类 | 罗列知识点 | 知识点 + 场景 + 应用 |
| 产品推广类 | 展示产品 | 产品 + 背后故事 + 改变 |
| 互动类 | 简单提问 | 问题 + 场景 + 价值 |
