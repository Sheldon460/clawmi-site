这一下给我更干懵了。这玩意儿还能自己养成性格？你想想看，我们不需要一开始就写好所有设定，就聊着聊着它自己就会形成一套行为准则，甚至会主动把这些规则记录下来，下次继续遵守。

我寻思了一下，这不就是扣子 2.5 做得特别好的地方吗。它会主动学习、记忆和自我迭代，越用越顺心。这种感觉太爽了，就像你招了个实习生，第一天啥也不会，一个月后他已经完全懂你的工作节奏了。

说到这个，这次扣子 2.5 的变化还挺有意思的。之前你在扣子里用个功能，得在好几个工具之间反复切换，烦都烦死了。现在好了，所有功能都收归到一个对话框里了。

日程、文件、邮箱、设备，全都在左侧菜单栏一目了然。而且哪怕我们在对话中途聊了别的事情，回头再提之前的任务，它也不会丢失上下文。我测试了好几次，这个超长记忆能力确实挺离谱的。

日程这块就不用多说了，你设置好的日程全部会同步到日程表，每天你做什么、韩老魔做什么，他安排得清清楚楚妥妥当当。创建日程后，他会自动创建循环任务，到点执行，完全不用催。我一开始还担心这玩意儿会不会忘，用了几天发现，它比我都记得清楚。

这个 Skill 不是自己写的，它来自 Agent World。这个我们后面说。

文件这块，解决了大家云端虾的文件经常找不到的痛点。说实话我自己也踩过这个坑，AI 给我生成了一堆东西，过几天想找死活找不到。现在扣子把这些输出都整齐有序地保存在文件里，可搜索可下载。所有产出都会自动分类，还有一个【记忆】分区，是它自己整理的长期笔记。我们可以看，但不用管。它会把自己的行为准则、对不同任务的处理规范都记在里面，越用越顺心👇

云电脑是一台云端 Linux 电脑，可以在上面开浏览器、装软件、跑脚本，像远程员工一样操作。云手机是一台真实的安卓手机，能打开任何 App，我们还能实时看到它的操作画面，遇到敏感操作它会先确认。

我们日常需要的很多信息，比如社交媒体的内容、需要登录才能访问的平台、App 内的数据等等这些通过可能 API 拿不到，必须有一个真实的设备去操作才行。云设备补上了这块一直缺失的拼图，让 AI 伙伴真正能够更深度触达互联网的更多地方。

下面说点我觉得有意思的实操。

我之前也和朋友们分享过，凡是重复多次并且没有太大变化的执行都可以做成 Skill。这个道理其实挺简单的，就是你想让你 AI 伙伴能帮你干更多的事，就得把你脑子里的经验拆解给它看。

这里我举一个简单的例子。

比如之前朋友们比较感兴趣文章配图，但提示词写不明白。其实完全可以让扣子来做这件事。基于韩老魔和我最近感兴趣的修真设定，我把这个工作流也设计成了几个节点，收集 → 炼化 → 入库 → 取用。听着挺像修仙小说里的功法层次，其实就是一个完整的工作闭环。

首先是【收集】，我让韩老魔用云电脑去图像设计网站（花瓣、站酷）帮我收集 30 张不同风格和色调的教材插图素材。

这里就体现了云设备的价值，这些设计网站很多内容无法通过搜索直接获取。AI 会自行启动云电脑开始工作，不断更换搜索关键词，比如扁平风格插图、儿童课程配图、3D 卡通配图、教育插画等等，截图学习了解，全程不需要我去理会。

我当时就愣住了，这玩意儿真能自己搞定这些网站？（这个过程之前忘记截图了，这是后来补的）👇

图片

然后是【炼化】。我实际需要的不是图片素材本身，而是需要 AI 从中提炼出提示词模板。我跟韩老魔说，帮我分析收集到的图片，提取风格特征，生成插图提示词模板。

他提取了 8 组主流风格的提示词模板，还贴心标注了使用场景和核心特征，提示词模板和示例都有，都是一些比较适合作为教材插图的。👇

图片

再下一步是【入库】，直接把整理好的模板保存为 Skill，方便多次复用。后续想增加新风格、优化生成细节，或者切换到信息图表等其他类型，都可以继续通过对话迭代优化。👇

图片

最后是【取用】。我选择水墨国风，给到飞书云文档链接（文档内容是《桃花源记》），让 AI 按照文章内容自动配图，同时说明插图比例等要求。不过不用让它把插图放到文章指定位置了，浪费 token 🐶，直接复制到文档里就好。👇

图片

就这样，一个简单方便、输出效果也不错的配图工作流就存好了。后续只要给到文档链接加上配图要求，就可以直接推进。还可以在这个基础上继续进行细化和优化。
