# 漫剧流水线项目

## 项目概述
自动化漫剧内容生成流水线，支持从脚本到成片的完整工作流。

## 目录结构

```
~/gemini-cli-projects/output/comic_drama/
├── pending/      # 待处理任务队列
├── processing/   # 正在处理的任务
├── completed/    # 已完成任务输出
├── assets/       # 公共素材库（模型、模板、字体等）
└── README.md     # 项目说明
```

## 使用说明

1. **添加任务**: 将任务配置放入 `pending/` 目录
2. **自动处理**: 流水线自动检测并开始处理
3. **查看结果**: 完成后输出到 `completed/` 目录
