## 经验记录

### [2026-03-08] SOUL.md 更新经验
- **场景**: 接到更新 SOUL.md 添加 Self-Improving 配置的指令
- **做法**: 
  1. 先读取现有 SOUL.md 了解当前结构
  2. 在文件开头添加 Self-Improving 配置区块
  3. 在 ACP 协议前添加任务启动记忆加载步骤
  4. 在群组协作协议后添加任务结束固化步骤
  5. 创建 self-improving/ 目录和初始化文件
- **效果**: 成功完成 SOUL.md 更新，建立个人记忆库结构
- **复用建议**: 其他 Agent 更新 SOUL.md 时可参考此流程

### [2026-03-08] 目录创建经验
- **场景**: 需要创建带空格的目录路径
- **做法**: 使用双引号包裹路径 `"/Volumes/My house/..."`
- **效果**: 成功创建目录，避免 Permission denied 错误
- **复用建议**: 所有涉及空格路径的操作都必须加引号
