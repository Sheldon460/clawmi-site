## 一、选题类提示词

### 1. 根据写作方向选题
```
我是一名公众号博主，我的写作方向是[领域]，请扮演[角色]，帮我列出这个话题下被广泛讨论的100个话题，我希望这些话题是人们在[场景]下会遇到的真实问题，请用简洁清晰的方式呈现。
```

**使用说明**：
- 替换[领域]：如职场、情感、育儿等
- 替换[角色]：如资深内容策划专家、行业专家等
- 替换[场景]：如工作、生活、学习等

**示例**：
```
我是一名公众号博主，我的写作方向是职场成长，请扮演资深职场导师，帮我列出这个话题下被广泛讨论的100个话题，我希望这些话题是人们在职场晋升过程中会遇到的真实问题，请用简洁清晰的方式呈现。
```

### 2. 根据热点+人群选题
```
你是一位资深的内容策划专家，你的任务是跟踪并分析当前关于<{热点内容}>的热点，为<{目标受众}>策划一系列的文章选题。你需要考虑这些热点对<{目标受众}>的吸引力，以及如何将热点主题融入到吸引人的选题中。请生成10个基于当前热点的文章选题。
```

**使用说明**：
- 替换<{热点内容}>：如AI技术、职场35岁危机、育儿焦虑等
- 替换<{目标受众}>：如职场新人、宝妈、学生等

**示例**：
```
你是一位资深的内容策划专家，你的任务是跟踪并分析当前关于<AI技术>的热点，为<职场新人>策划一系列的文章选题。你需要考虑这些热点对<职场新人>的吸引力，以及如何将热点主题融入到吸引人的选题中。请生成10个基于当前热点的文章选题。
```

### 3. 根据人群痛点选题
```
你是一位资深的内容策划专家，你的任务是为<{目标受众}>策划一系列的文章选题。你需要考虑这些话题能切中他们的痛点，引起情绪共鸣。请生成10个基于当前热点的文章选题。
```

**使用说明**：
- 替换<{目标受众}>：如职场新人、宝妈、学生等

**示例**：
```
你是一位资深的内容策划专家，你的任务是为<职场新人>策划一系列的文章选题。你需要考虑这些话题能切中他们的痛点，引起情绪共鸣。请生成10个基于当前热点的文章选题。
```

## 二、标题类提示词

### 1. 标题生成提示词
```
你是一名资深自媒体写手，现在你要为我主题<xxx>的文章写标题。在写标题时，可以使用以下技巧并参考范例:

1、巧用数字
数字标题给人清晰、具体、易操作的感觉，能快速吸引读者的注意力。
范例:这5个早晨习惯，让我效率提升300%。

2、场景代入
细节场景触发"心理模拟"，增强用户的代入感
范例:为什么你的努力总是得不到回报？原因可能只有这一个。

3、制造对比
对比冲突制造认知失衡，刺激用户的点击欲。
范例:我月入两万却活得像月光族，直到发现了这个秘密。

4、制造悬念
就像没有人不爱看悬疑小说一样，悬念能够很好的激发好奇心。
范例:当我打开那个神秘文件夹，才明白自己错过了什么。

5.贴标签
强化身份认同，让读者感觉"说的就是我"
范例: 劝你千万别做"老实人"：职场晋升必备的3个心机技巧

6、名人效应
名人之所以是名人，就是因为他们自带流量，适当蹭一蹭，可以提升你的打开率。
范例:李佳琦直播从不推荐的护肤品，却是他自己每天都在用。

每个技巧给出5个标题，共30个，只给标题，不要做任何解释。
```

**使用说明**：
- 替换<xxx>：你的文章主题
- 技巧说明：数字、场景代入、制造对比、设置悬念、贴标签、名人效应
- 输出要求：每个技巧5个标题，共30个，只给标题

### 2. 标题改写提示词
```
现在标题是<xxx>。请你分析这个标题的爆款要素,并参考标题的风格、句式和字数,给我三个关于xxx多角度但同类型的标题，略带口语化。
```

**使用说明**：
- 替换<xxx>：原标题
- 技巧说明：分析爆款要素，参考风格、句式和字数
- 输出要求：3个同类型标题，略带口语化

## 三、开头类提示词

### 1. 开头生成提示词
```
你是一名资深自媒体写手，现在你要为我主题大纲<xxx>的文章写开头。在写开头时，可以使用以下技巧并参考范例:

1. 用提问开头：
用提问开头，直击目标读者的日常困扰，用连续三问营造共鸣，让读者感觉"这就是在我说"。
示例："每天24小时都不够用？
手机提醒响个不停，工作永远做不完？
周末本该休息，却还在处理各种杂事？"

2. 用故事开头：
开头用故事，反转制造悬念，引发读者好奇心。
示例："昨天下班回家，我遇到了一个送外卖的小哥。
聊天中得知，他去年还是某知名互联网公司的程序员，年薪50万+。
为什么他会选择放弃高薪工作？他说的这句话让我沉思许久..."

3. 用名言开头：
利用名人效应增加文章可信度，加入个人观点和经历，用反转吸引注意力。
示例："乔布斯说：'保持饥饿，保持愚蠢。'
这句话我曾经觉得特别矫情。
直到上个月我裁员后，才真正懂得了这句话的分量..."

4. 用场景开头：
通过某个场景开头，让读者身临其境。
示例："凌晨两点，办公室只剩下屏幕的蓝光。
你揉了揉酸痛的脖子，看着面前写了一半的报告。
手机响了，是老板的消息：'这个方案明早8点前必须发给客户...'"

开头要简短，200字以内，根据以上范例各写一个开头，只给开头，不要做任何解释。
```

**使用说明**：
- 替换<xxx>：文章主题大纲
- 技巧说明：提问、故事、名言、场景
- 输出要求：200字以内，各写一个开头，只给开头

## 五、结尾类提示词

### 1. 结尾生成提示词
```
你是一名资深的自媒体写手，你擅长写出能够有效吸引读者注意力的文章，请为主题为xxx，受众人群为xxx的文章写结尾，可以参考以下范例：

1、总结式结尾:
这种模板适用于强调文章的主要论点，通过简洁明了地重申文章中的关键信息来结束文章。
示例:"综上所述，有效管理时间不仅能提高工作效率，还能增加个人的幸福感。实施我们讨论的策略,开始享受更有组织的生活吧。"

2、行动号召结尾:
结尾处提供具体的行动号召，鼓励读者根据文章内容采取行动。
示例:未来淘汰你的不是AI，是会使用AI的人，对于我们普通人来说，最重要的，就是把AI工具用起来把一些适合AI的工作交给它们，未来已来，不要错过！

3、提问结尾:
通过提问结束文章，激发读者思考，使他们对文章内容进行更深入的反思。
示例:要写的内容到这里，但我深知，光是阅读只能给你造成“我已经学会了”的假象，更重要的是实践.是练习，所以，你准备如何开始呢?

4、名言结尾:
使用相关的名言或引用作为文章的结尾，增加文章的权威性和深度，注意名人名言的准确性。
示例:"正如罗斯福所说，"唯一值得恐惧的，就是恐惧本身。"面对挑战，让我们以积极的态度迎接每一个困难。

根据以上范例各写一个结尾，200字以内，只写结尾，不要做任何解释。
```

**使用说明**：
- 替换<xxx>：文章主题
- 替换<xxx>：受众人群
- 技巧说明：总结式、行动号召、提问、名言
- 输出要求：200字以内，各写一个结尾，只写结尾

## 七、拆解类提示词

### 1. 爆文拆解提示词
```
请你作为一名内容分析专家，对我提供的文章进行深度拆解。请从以下维度展开分析，确保拆解全面且可操作：

1. 对标账号分析
   - 账号的核心定位是什么？
   - 目标受众群体画像如何？
   - 账号的独特竞争力或差异化优势是什么？

2. 标题拆解
   - 标题属于哪种类型？（如：数字型、疑问型、对比型、悬念型等）
   - 核心关键词和吸引力点是什么？
   - 标题如何引发读者兴趣或情感共鸣？

3. 开头设计
   - 开篇采用了什么技巧？（如：故事引入、数据冲击、问题引导等）
   - 如何在3秒内抓住读者注意力？
   - 开头如何引出文章核心观点？

4. 内容结构
   - 文章的整体框架是什么？（如：总分总、问题-解决方案、故事-启示等）
   - 各部分之间如何衔接和过渡？
   - 论述逻辑是否清晰？层次感如何？

5. 结尾设计
   - 结尾采用了什么方式？（如：总结升华、号召行动、情感共鸣等）
   - 如何实现文章的情感升华或价值传递？
   - 是否有明确的号召性动作？

6. 语言风格
   - 整体语气特点是什么？（如：幽默、专业、亲切等）
   - 是否有独特的表达方式或高频词汇？
   - 如何通过语言风格增强文章的吸引力？

7. 人群定位
   - 目标读者的核心痛点和需求是什么？
   - 文章如何与目标读者建立情感连接？
   - 内容是否精准匹配了目标人群的偏好？

8. 文章金句
   - 提取3-5个最具传播力的金句。
   - 分析这些金句的创作技巧和打动人心之处。

9. 总结
   - 这篇文章成功的关键要素是什么？
   - 有哪些可复制的创作技巧？
   - 如何将这些技巧应用到自己的内容创作中？

【文章内容】：
```

**使用说明**：
- 技巧说明：从9个维度深度拆解
- 输出要求：全面且可操作

## 八、二创类提示词

### 1. 小绿书二创提示词
```
## Role
你是一位经验丰富的公众号小绿书创作者，擅长将长篇文章精炼成吸引人的短文内容，具备敏锐的内容洞察力和强大的文案创作能力，能够快速抓住文章精髓并转化为适合社交媒体传播的优质内容。

## Task
- 根据我提供的文章链接 （插入链接地址） ，深度分析原文内容和结构
- 提炼文章的核心观点和关键信息
- 创作一篇300-500字的小绿书文章，确保内容既保留原文精髓又具备强烈的阅读吸引力
- 设计引人入胜的开头和促进互动的结尾

## Skills

### 技能1：内容分析与提炼
1. 仔细阅读原文章，识别核心主题和目标受众
2. 根据原文内容确定适合的写作风格（专业、轻松、情感等）
3. 概括性地涵盖文章的多个重要观点，确保信息完整性
4. 提取最具价值的案例、数据或故事素材

### 技能2：吸引力开头设计
1. 前三行必须具备强烈的吸引力，可采用以下方式：
   - 抛出引人思考的问题
   - 使用反常识或令人意外的观点
   - 描述生动的场景或情境
   - 引用震撼的数据或事实
2. 确保开头与原文主题紧密相关
3. 营造悬念，激发读者继续阅读的欲望

### 技能3：结构化内容创作
按照"开头吸引+核心观点+案例说明+互动结尾"的结构：
1. 开头吸引 （50-80字）：设计吸睛的前三行
2. 核心观点 （150-250字）：概括性呈现原文的主要观点
3. 案例说明 （80-120字）：选择原文中最有说服力的案例或数据
4. 互动结尾 （30-50字）：提出开放性问题引导读者思考和留言

### 技能4：互动引导设计
- 根据文章主题设计相关的开放性问题
- 问题要能激发读者分享个人经历或观点
- 使用"你觉得...？"、"你遇到过...吗？"等句式
- 确保问题与文章内容高度相关

## Constraint

### 输出规范
- 严格控制字数在300-500字之间
- 使用适合社交媒体阅读的短句和段落
- 保持原文的核心观点和价值导向
- 文章结构清晰，逻辑流畅

### 行为边界
- 不得偏离原文的主要观点和事实
- 不添加原文中没有的虚假信息
- 保持内容的准确性和客观性
- 适应原文的目标受众群体

### 质量标准
- 开头三行必须具备强烈吸引力
- 内容要有价值，能给读者带来启发或收获
- 语言表达要符合原文确定的风格调性
- 互动问题要自然融入，不显突兀
- 整体内容要具备传播价值和分享欲望
```

**使用说明**：
- 技巧说明：角色扮演+任务+技能+约束
- 输出要求：300-500字，结构清晰，互动性强

### 2. 小红书爆款二创提示词
```
请将【原文】小红书笔记改写为200-500字左右的带货文案，要求：

【背景】 公众号带货文案需要兼具亲和力和引导性，通过文字打动目标用户，激发购买欲望，并实现转化。

【任务】 根据产品信息和目标用户需求，撰写结构清晰、内容吸引的文案，突出产品价值，明确传递购买指引。

【要求】 文案包含吸睛标题、引发共鸣的开篇,产品优势和应用场景的详细介绍，以及带有紧迫感的购买逼单话术。语言亲和，内容贴近用户需求强调产品的实际价值和使用场景，适当使用emoji表情。

【原文】：[粘贴小红书笔记内容]
```

**使用说明**：
- 技巧说明：背景+任务+要求
- 输出要求：200-500字，结构清晰，带货导向

## 使用建议

### 1. 提示词优化原则
- **明确角色**：让AI扮演特定角色，如"资深自媒体写手"、"内容策划专家"
- **具体任务**：明确告诉AI要做什么，如"生成标题"、"写开头"
- **清晰要求**：指定输出格式、字数、风格等
- **提供范例**：给出参考示例，帮助AI理解期望
- **限制输出**：明确要求"只给标题"、"不要解释"等

### 2. 提示词使用技巧
- **分步使用**：先选题，再标题，再开头，再正文，再结尾
- **迭代优化**：根据AI输出结果，调整提示词进行优化
- **结合素材**：提供具体素材，让AI生成更贴合的内容
- **角色扮演**：让AI扮演特定角色，提升输出质量

### 3. 提示词组合使用
- **选题+标题**：先生成选题，再针对选题生成标题
- **标题+开头**：先生成标题，再根据标题生成开头
- **开头+正文**：先生成开头，再根据开头生成正文
- **正文+结尾**：先生成正文，再根据正文生成结尾

### 4. 提示词调整策略
- **如果AI输出太泛**：增加具体限制，如"字数控制在300字以内"
- **如果AI输出太浅**：增加深度要求，如"深入分析"、"提供具体案例"
- **如果AI输出不符合风格**：明确风格要求，如"使用幽默语言"、"保持专业性"
- **如果AI输出太长**：明确字数限制，如"200字以内"、"1000字左右"
