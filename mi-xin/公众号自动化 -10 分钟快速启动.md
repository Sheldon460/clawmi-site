# 🚀 微信公众号自动化 - 10 分钟快速启动

## 前置条件

- ✅ 云服务器（Ubuntu 22.04+）
- ✅ 微信公众号（认证服务号）
- ✅ OpenClaw 已安装

## 5 步配置

### 1️⃣ 安装技能（2 分钟）

```bash
cd ~/.openclaw/skills
git clone https://github.com/JimLiu/canghe-skills.git
```

### 2️⃣ 配置 API 凭证（2 分钟）

```bash
# 登录公众号后台获取
# https://mp.weixin.qq.com → 开发 → 基本配置

cat > ~/.canghe-skills/.env << 'EOF'
WECHAT_APP_ID=你的 AppID
WECHAT_APP_SECRET=你的 AppSecret
WECHAT_BROWSER_CHROME_PATH=/usr/bin/google-chrome
default_publish_method: api
EOF
```

### 3️⃣ 配置 IP 白名单（1 分钟）

公众号后台 → 开发 → 基本配置 → IP 白名单  
添加你的云服务器公网 IP

### 4️⃣ 测试 API（1 分钟）

```bash
curl -s "https://api.weixin.qq.com/cgi-bin/token?grant_type=client_credential&appid=你的 AppID&secret=你的 AppSecret" | jq .
```

看到 `access_token` 即成功 ✅

### 5️⃣ 发布第一篇文章（4 分钟）

```bash
# 准备文章
cat > ~/article.md << 'EOF'
---
title: 我的第一篇文章
author: 作者名
---

# 我的第一篇文章

这是内容...
EOF

# 转换 HTML
cd ~/.openclaw/skills/canghe-markdown-to-html
npx -y bun scripts/main.ts ~/article.md --theme simple

# 发布
cd ../canghe-post-to-wechat
npx -y bun scripts/wechat-api.ts ~/article.html
```

## ✅ 完成！

查看草稿：https://mp.weixin.qq.com
