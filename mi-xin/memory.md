## 关键坑记录 (Critical Pitfalls)

### 🚨 坑 1：浏览器发布方式已弃用
- **日期**: 2026-03-10
- **原因**: 富文本编辑器粘贴不稳定，成功率仅 50%
- **症状**: `Body content verification failed: editor appears empty after paste`
- **解决方案**: 使用 API 发布方式
- **参考**: 迭代分析报告 3.2 节

### 🚨 坑 2：IP 白名单是最常见错误
- **日期**: 2026-03-10
- **错误码**: `40164 - invalid ip, not in whitelist`
- **发生频率**: 新手 100%，老手 20%
- **解决方案**: 公众号后台 → 开发 → 基本配置 → IP 白名单
- **预防措施**: 使用配置检查脚本

### 🚨 坑 3：humanize-zh 技能未安装
- **日期**: 2026-03-10
- **影响**: 文章风格化需手动 L2 处理
- **临时方案**: 手动进行 L2 中度处理
- **解决方案**: 安装 humanize-zh 技能并集成到创作流程
- **计划解决**: V2.1
