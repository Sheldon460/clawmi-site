## 2. API 凭证配置

### 2.1 获取微信公众号 API 凭证

**步骤 1**: 登录微信公众号后台
```
https://mp.weixin.qq.com
```

**步骤 2**: 进入开发设置
```
左侧菜单 → 开发 → 基本配置
```

**步骤 3**: 获取凭证
- **AppID**: 开发者 ID（如：wx1234567890abcdef）
- **AppSecret**: 开发者密码（如：1234567890abcdef1234567890abcdef）

**步骤 4**: 配置 IP 白名单（重要！）
```
在"基本配置"页面找到"IP 白名单"
添加你的云服务器公网 IP 地址
```

⚠️ **注意**: 不配置 IP 白名单会导致 API 调用失败

### 2.2 测试 API 凭证

```bash
# 创建测试脚本
cat > /tmp/test-wechat-api.sh << 'EOF'
#!/bin/bash

APP_ID="你的 AppID"

# 获取 access_token
TOKEN_URL="https://api.weixin.qq.com/cgi-bin/token?grant_type=client_credential&appid=${APP_ID}&secret=${APP_SECRET}"

echo "API 响应:"
echo $RESPONSE | jq .

# 检查是否成功
```

### 2.3 获取 Access Token（自动化脚本）

```bash
# 创建 token 获取脚本
cat > ~/openclaw-workspace/scripts/get-wechat-token.sh << 'EOF'
#!/bin/bash

ENV_FILE="$HOME/.canghe-skills/.env"
TOKEN_FILE="$HOME/.canghe-skills/wechat-token.json"

# 读取配置
APP_ID=$(grep WECHAT_APP_ID $ENV_FILE | cut -d'=' -f2)
APP_SECRET=$(grep WECHAT_APP_SECRET $ENV_FILE | cut -d'=' -f2)

# 获取 token
TOKEN_URL="https://api.weixin.qq.com/cgi-bin/token?grant_type=client_credential&appid=${APP_ID}&secret=${APP_SECRET}"
RESPONSE=$(curl -s $TOKEN_URL)

# 保存 token
echo $RESPONSE > $TOKEN_FILE

# 提取 token
ACCESS_TOKEN=$(echo $RESPONSE | jq -r '.access_token')
EXPIRES_IN=$(echo $RESPONSE | jq -r '.expires_in')

if [ "$ACCESS_TOKEN" != "null" ]; then
    echo "✅ Token 获取成功"
    echo "Access Token: $ACCESS_TOKEN"
    echo "有效期：$EXPIRES_IN 秒"
    echo "过期时间：$(date -d "+$EXPIRES_IN seconds" '+%Y-%m-%d %H:%M:%S')"
else
    echo "❌ Token 获取失败"
    echo $RESPONSE | jq .
    exit 1
fi
EOF

chmod +x ~/openclaw-workspace/scripts/get-wechat-token.sh
```

### 3.2 批量抓取素材

```bash
# 创建批量抓取脚本
cat > ~/openclaw-workspace/scripts/batch-fetch-urls.sh << 'EOF'
#!/bin/bash

# 文章链接列表（每行一个 URL）
URLS=(
    "https://mp.weixin.qq.com/s/xxxxx1"
    "https://mp.weixin.qq.com/s/xxxxx2"
    "https://mp.weixin.qq.com/s/xxxxx3"
)

OUTPUT_DIR="$HOME/openclaw-workspace/url-content"
mkdir -p $OUTPUT_DIR

SKILL_DIR="$HOME/.openclaw/skills/canghe-url-to-markdown"

echo "开始批量抓取 ${#URLS[@]} 篇文章..."

for i in "${!URLS[@]}"; do
    url="${URLS[$i]}"
    echo "[$((i+1))/${#URLS[@]}] 抓取：$url"
    
    # 生成输出文件名
    output_file="$OUTPUT_DIR/article-$((i+1)).md"
    
    # 执行抓取
    export WECHAT_BROWSER_CHROME_PATH=/usr/bin/google-chrome
    npx -y bun $SKILL_DIR/scripts/main.ts "$url" -o "$output_file"
    
    if [ $? -eq 0 ]; then
        echo "✅ 抓取成功：$output_file"
    else
        echo "❌ 抓取失败：$url"
    fi
done

echo "批量抓取完成！"
echo "输出目录：$OUTPUT_DIR"
EOF

chmod +x ~/openclaw-workspace/scripts/batch-fetch-urls.sh
```

```bash
    echo "⚠️ 未找到相关素材，请先抓取素材"
```

## 6. 自动发布流程

### 6.1 API 发布脚本

```bash
# 创建 API 发布脚本
cat > ~/openclaw-workspace/scripts/publish-to-wechat.sh << 'EOF'
#!/bin/bash

set -e

# 配置文件
ENV_FILE="$HOME/.canghe-skills/.env"
TOKEN_FILE="$HOME/.canghe-skills/wechat-token.json"

# 读取配置
APP_ID=$(grep WECHAT_APP_ID $ENV_FILE | cut -d'=' -f2)
APP_SECRET=$(grep WECHAT_APP_SECRET $ENV_FILE | cut -d'=' -f2)

# 获取 token
echo "获取 access_token..."
TOKEN_URL="https://api.weixin.qq.com/cgi-bin/token?grant_type=client_credential&appid=${APP_ID}&secret=${APP_SECRET}"
TOKEN_RESPONSE=$(curl -s $TOKEN_URL)
ACCESS_TOKEN=$(echo $TOKEN_RESPONSE | jq -r '.access_token')

if [ "$ACCESS_TOKEN" == "null" ]; then
    echo "❌ 获取 token 失败"
    echo $TOKEN_RESPONSE | jq .
    exit 1
fi

echo "✅ Token 获取成功"

# 参数
HTML_FILE="$1"
TITLE="$2"

if [ -z "$HTML_FILE" ] || [ -z "$TITLE" ]; then
    echo "用法：$0 <html 文件路径> <文章标题>"
    echo "示例：$0 ./article.html '文章标题'"
    exit 1
fi

# 读取 HTML 内容
HTML_CONTENT=$(cat $HTML_FILE)

# 获取封面图 ID（需要先上传图片）
echo "上传封面图..."
# 这里需要调用 media/uploadimg API

# 创建草稿
echo "创建草稿..."
CREATE_URL="https://api.weixin.qq.com/cgi-bin/draft/add?access_token=${ACCESS_TOKEN}"

# 构建 JSON 数据（简化版）
JSON_DATA=$(cat << EOF
{
    "articles": [
        {
            "title": "$TITLE",
            "content": "$HTML_CONTENT",
            "thumb_media_id": "封面图 ID",
            "author": "作者名",
            "need_open_comment": 1,
            "only_fans_can_comment": 0
        }
    ]
}
EOF
)

# 发送请求
RESPONSE=$(curl -s -X POST $CREATE_URL -d "$JSON_DATA")

echo "API 响应:"
echo $RESPONSE | jq .

# 检查结果
if echo $RESPONSE | jq -e '.media_id' > /dev/null; then
    MEDIA_ID=$(echo $RESPONSE | jq -r '.media_id')
    echo "✅ 发布成功！"
    echo "Media ID: $MEDIA_ID"
    echo "查看草稿：https://mp.weixin.qq.com"
else
    echo "❌ 发布失败"
    echo $RESPONSE | jq .
    exit 1
fi
EOF

chmod +x ~/openclaw-workspace/scripts/publish-to-wechat.sh
```

## 7. 问题排查

### 7.1 常见问题 FAQ

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| API 调用失败 | IP 白名单未配置 | 在公众号后台添加服务器 IP |
| Token 获取失败 | AppID/Secret 错误 | 检查凭证是否正确复制 |
| Chrome 找不到 | 路径配置错误 | 设置 WECHAT_BROWSER_CHROME_PATH |
| 粘贴失败 | 富文本编辑器拦截 | 使用 API 方式发布 |
| 图片上传失败 | 文件大小超限 | 压缩图片到 2MB 以内 |
| 发布失败 | 内容违规 | 检查敏感词 |
