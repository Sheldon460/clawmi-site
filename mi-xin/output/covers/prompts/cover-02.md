# Cover Image Prompt

## 主题与概念
AI正在摧毁互联网流量入口 - 视觉冲击力强，展现入口崩塌的隐喻，新旧时代交替的危机感与变革感。

## 视觉元素
- **主视觉**: 抽象的"入口"或"门"正在崩塌、碎裂的场景
- **构图**: 动态撕裂感，旧的结构在崩塌，新的光芒穿透
- **氛围**: 危机感、戏剧性、变革的冲击力

## 色彩方案
- **暗红**: #991B1B, #DC2626 (危机、警示)
- **黑色**: #0A0A0A, #1F1F1F (深邃、未知)
- **点缀光**: #FCD34D (变革的希望)
- **背景**: 深黑到暗红渐变

## 必须包含的元素
1. 破碎/崩塌的几何形状（象征入口）
2. 动态粒子或碎片效果
3. 光线从裂缝中透出的隐喻
4. 危机感与希望感并存的对比

## 构图要求
- 40-60%留白
- 主元素居中或偏右
- 动态对角线构图增强冲击力
- 标题文字区域预留（左下）
