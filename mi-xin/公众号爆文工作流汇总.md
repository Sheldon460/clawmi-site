## 一、AI写作工作流

### 1. 完整AI写作工作流

```
选题 → 标题 → 开头 → 正文 → 结尾 → 金句 → 封面图 → 关键词部署 → 敏感词检查 → 原创度检测 → 发布
```

### 2. 分步工作流详解

#### 第1步：选题策划
**目标**：确定文章主题，确保选题具有爆文潜力

**操作流程**：
1. **确定写作方向**：明确领域（职场、情感、育儿等）
2. **选择选题方法**：
   - 根据写作方向选题
   - 根据热点+人群选题
   - 根据人群痛点选题
3. **验证选题潜力**：
   - 查看微信指数
   - 分析对标账号爆款
   - 评估普适性、痛点、共鸣度、讨论度、新知度

**AI提示词**：
```
我是一名公众号博主，我的写作方向是[领域]，请扮演[角色]，帮我列出这个话题下被广泛讨论的100个话题，我希望这些话题是人们在[场景]下会遇到的真实问题，请用简洁清晰的方式呈现。
```

#### 第2步：标题创作
**目标**：打造高点击率标题

**操作流程**：
1. **选择标题技巧**（6选1或组合）：
   - 巧用数字
   - 场景代入
   - 制造对比
   - 设置悬念
   - 贴标签
   - 名人效应
2. **生成标题**：使用AI生成30个标题
3. **筛选优化**：选择最吸引人的3-5个标题
4. **改写优化**：对优选标题进行口语化改写

**AI提示词**：
```
你是一名资深自媒体写手，现在你要为我主题<xxx>的文章写标题。在写标题时，可以使用以下技巧并参考范例: [6个技巧说明] 每个技巧给出5个标题，共30个，只给标题，不要做任何解释。
```

#### 第3步：开头设计
**目标**：3秒内抓住读者注意力

**操作流程**：
1. **选择开头方式**（4选1）：
   - 提问开头
   - 故事开头
   - 名言开头
   - 场景开头
2. **生成开头**：使用AI生成4个开头
3. **选择优化**：选择最吸引人的开头
4. **调整字数**：控制在200字以内

**AI提示词**：
```
你是一名资深自媒体写手，现在你要为我主题大纲<xxx>的文章写开头。在写开头时，可以使用以下技巧并参考范例: [4个技巧说明] 开头要简短，200字以内，根据以上范例各写一个开头，只给开头，不要做任何解释。
```

#### 第4步：正文创作
**目标**：提供有价值的内容，结构清晰

**操作流程**：
1. **分析文章主题和受众**：明确写作目标
2. **选择文章结构**（3选1）：
   - 总分总结构
   - 金字塔结构
   - SCQA结构
3. **准备素材**：收集相关案例、数据、故事
4. **生成正文**：使用AI生成1200字左右正文
5. **优化调整**：检查逻辑、流畅度、价值感

**AI提示词**：
```
你是一名资深的自媒体写手，你擅长写出能够有效吸引读者注意力的文章，请先分析文章主题xxx，受众人群为xxx，结合提供的素材，从3种文章结构（总分总、金字塔、SCQA）选择合适的文章结构：

素材1：
素材2：
素材3：

实用**语言风格，写一篇1200字以内的正文，不要做任何解释。
```

#### 第5步：结尾设计
**目标**：给读者留下深刻印象，引导行动

**操作流程**：
1. **选择结尾方式**（4选1）：
   - 总结式结尾
   - 名言式结尾
   - 号召式结尾
   - 提问式结尾
2. **生成结尾**：使用AI生成4个结尾
3. **选择优化**：选择最合适的结尾
4. **调整字数**：控制在200字以内

**AI提示词**：
```
你是一名资深的自媒体写手，你擅长写出能够有效吸引读者注意力的文章，请为主题为xxx，受众人群为xxx的文章写结尾，可以参考以下范例: [4个技巧说明] 根据以上范例各写一个结尾，200字以内，只写结尾，不要做任何解释。
```

#### 第6步：金句总结
**目标**：提炼文章金句，增强传播力

**操作流程**：
1. **阅读全文**：理解文章核心观点
2. **提取金句**：找出3-5个最具传播力的句子
3. **优化金句**：确保短小精悍、朗朗上口、引发共鸣
4. **插入位置**：将金句插入文章关键位置

**AI提示词**：
```
你是一名资深的自媒体写手，你擅长写出能够引发读者情绪共鸣的文章，请为我的文章总结3个金句，要求短小精悍、朗朗上口、引发情绪共鸣！直接输出金句，不需任何解释。

【文章内容】
```

#### 第7步：封面图生成
**目标**：制作吸引眼球的封面图

**操作流程**：
1. **提炼文章主题**：用1句话概括文章核心
2. **生成描述**：转化为文生图指令
3. **生成图片**：使用即梦AI生成封面图
4. **裁剪优化**：调整为3:2比例，去除水印

**AI提示词**：
```
你是一名文生图指令专家，请根据我提供的【文章内容】，总结归纳文章概要，并输出即梦文生图的指令。
图片比例选择3:2,这个比例方便封面图裁切的时候，还预留了少量的裁切空间。
```

#### 第8步：关键词部署
**目标**：提高文章搜索曝光

**操作流程**：
1. **关键词挖掘**：
   - 使用微信指数
   - 使用5118长尾挖掘
   - 使用新榜爆文分析
2. **关键词部署**：
   - 标题中部署核心关键词
   - 正文中自然融入关键词
   - 小标题中融入关键词
   - 使用长尾关键词
3. **标签设置**：设置3-5个精准标签
4. **字数控制**：控制在1200字左右（广告位需求）

#### 第9步：敏感词检查
**目标**：避免违规，确保内容安全

**操作流程**：
1. **使用工具**：96编辑器违规检测系统
2. **检查维度**：
   - 暴恐违禁
   - 政治敏感
   - 低俗用语
   - 色情暗示
   - 低质灌水
3. **修改优化**：替换或删除敏感词
4. **二次检查**：发布前再次检查

#### 第10步：原创度检测
**目标**：降低AI痕迹，提高原创度

**操作流程**：
1. **使用工具**：朱雀AI检测工具
2. **检测标准**：AI生成度控制在40%以下
3. **优化调整**：
   - 增加个人观点
   - 调整表达方式
   - 补充个人经历
4. **二次检测**：优化后再次检测

#### 第11步：发布前检查
**目标**：确保内容质量，避免发布后问题

**检查清单**：
- [ ] 标题是否吸引人
- [ ] 开头是否抓人
- [ ] 正文结构是否清晰
- [ ] 结尾是否有价值
- [ ] 金句是否突出
- [ ] 封面图是否美观
- [ ] 关键词是否部署
- [ ] 敏感词是否清除
- [ ] 原创度是否达标
- [ ] 标签是否精准

#### 第12步：发布
**目标**：选择最佳时间发布，获得最大曝光

**操作流程**：
1. **选择发布时间**：根据领域选择最佳时段
2. **选择发布方式**：
   - 单篇发布
   - 多篇合并发布
3. **设置推荐**：确保平台推荐开关打开
4. **预览检查**：手机端预览效果
5. **定时发布**：可设置定时发布

## 二、小绿书变现SOP

### 1. 小绿书发布工作流

```
选题 → 文案创作 → 图片制作 → 带货链接 → 发布 → 评论区运营
```

### 2. 分步工作流详解

#### 第1步：选题策划
**目标**：选择适合小绿书的爆款选题

**操作流程**：
1. **找对标**：在"看一看"中找对标账号
2. **选爆款**：选择点赞>50或阅读>1000的爆款
3. **定方向**：确保与公众号赛道一致
4. **准备素材**：收集相关图片、文案

#### 第2步：文案创作
**目标**：创作200-500字的优质文案

**操作流程**：
1. **选择创作方式**：
   - 纯文字版：直接创作
   - 二创版：基于爆款内容二创
2. **使用AI辅助**：使用KIMI、DeepSeek等工具
3. **优化文案**：确保口语化、有吸引力
4. **检查字数**：控制在200-500字

**AI提示词（二创）**：
```
## Role
你是一位经验丰富的公众号小绿书创作者，擅长将长篇文章精炼成吸引人的短文内容...

## Task
- 根据我提供的文章链接，深度分析原文内容和结构
- 提炼文章的核心观点和关键信息
- 创作一篇300-500字的小绿书文章...
```

#### 第3步：图片制作
**目标**：制作吸引眼球的图片

**操作流程**：
1. **选择图片类型**：
   - 纯文字版：无需图片
   - 图文版：准备1-20张图片
2. **图片处理**：
   - 使用搞定设计、美图秀秀等工具
   - 批量处理图片（裁剪、加边框、加滤镜）
   - 修改MD5值，避免搬运检测
3. **封面图制作**：制作3:2比例封面图

#### 第4步：带货链接
**目标**：添加带货商品，实现变现

**操作流程**：
1. **开通带货功能**：
   - 100粉丝：开通返佣商品功能
   - 500粉丝：开通流量主
2. **选品**：
   - 选择高佣商品（30%-50%）
   - 选择与内容相关商品
   - 查看商品评分和佣金率
3. **添加链接**：在编辑器中插入商品链接
4. **检查合规**：避免过度营销

#### 第5步：发布
**目标**：选择最佳时间发布

**操作流程**：
1. **选择发布方式**：
   - 纯文字版：直接发布
   - 图文版：上传图片+文案
2. **设置标签**：添加3-5个精准标签
3. **选择时间**：根据领域选择最佳时段
4. **预览检查**：手机端预览效果
5. **发布**：点击发布按钮

#### 第6步：评论区运营
**目标**：提升互动，增加广告收益

**操作流程**：
1. **自己留言**：发布后自己先留言5条
2. **插入链接**：在评论区插入已发文章链接
3. **回复互动**：及时回复用户评论
4. **引导话题**：引导用户讨论相关话题
5. **监控数据**：关注阅读量、点赞、评论数据

## 三、文章运营工作流

### 1. 文章发布工作流

```
选题 → 写作 → 检查 → 发布 → 评论管理 → 数据分析 → 优化迭代
```

### 2. 分步工作流详解

#### 第1步：选题策划
**目标**：选择有爆文潜力的选题

**操作流程**：
1. **找对标账号**：寻找低粉爆文账号
2. **分析爆款**：拆解对标账号爆款文章
3. **确定选题**：选择普适性高、痛点强、有讨论度的选题
4. **验证潜力**：使用微信指数验证关键词热度

#### 第2步：内容创作
**目标**：创作高质量文章

**操作流程**：
1. **使用AI写作**：按照AI写作工作流创作
2. **人工优化**：增加个人观点、个人经历
3. **检查质量**：确保内容有价值、有深度
4. **降低AI味**：调整表达方式，增加人性化内容

#### 第3步：发布前检查
**目标**：确保内容合规、优质

**操作流程**：
1. **敏感词检查**：使用96编辑器检测
2. **原创度检测**：使用朱雀检测，控制AI生成度在40%以下
3. **关键词部署**：检查关键词是否自然融入
4. **标签设置**：设置3-5个精准标签
5. **字数检查**：控制在1200字左右

#### 第4步：发布
**目标**：选择最佳时间发布

**操作流程**：
1. **选择发布时间**：根据领域选择最佳时段
2. **选择发布方式**：
   - 单篇发布
   - 多篇合并发布（最多8篇）
3. **设置推荐**：确保平台推荐开关打开
4. **预览检查**：手机端预览效果
5. **发布**：点击发布按钮

#### 第5步：评论管理
**目标**：提升互动，控制话题走向

**操作流程**：
1. **人工审核**：审核所有留言，隐藏负面内容
2. **精选留言**：选择优质留言展示
3. **引导话题**：通过精选留言引导讨论方向
4. **回复互动**：及时回复用户评论
5. **监控舆情**：避免敏感词和负面情绪扩散

#### 第6步：数据分析
**目标**：了解文章表现，发现优化点

**操作流程**：
1. **查看阅读量**：关注阅读量变化趋势
2. **分析来源**：查看阅读来源（推荐、分享、搜索等）
3. **查看互动**：分析点赞、在看、评论数据
4. **识别异常值**：发现阅读量异常高的文章
5. **总结规律**：总结爆款文章的共同点

#### 第7步：优化迭代
**目标**：持续优化，提高爆文率

**操作流程**：
1. **分析爆款**：深入分析异常值文章
2. **复制选题**：围绕爆款选题继续创作
3. **调整策略**：根据数据反馈调整写作策略
4. **持续更新**：保持稳定更新频率
5. **建立体系**：形成自己的内容体系

## 四、账号运营工作流

### 1. 账号起号工作流

```
注册 → 养号 → 定位 → 找对标 → 写作 → 发布 → 数据分析 → 优化迭代
```

### 2. 分步工作流详解

#### 第1步：账号注册
**目标**：快速注册公众号

**操作流程**：
1. **准备材料**：身份证、实名微信、邮箱
2. **填写信息**：名称、简介、类目
3. **身份验证**：扫码刷脸验证
4. **设置功能**：欢迎语、菜单栏
5. **下载APP**：下载订阅号助手

#### 第2步：账号养号
**目标**：让账号进入流量池

**操作流程**：
1. **养号3天**：不发文章，只存草稿
2. **发Python文章**：发12-15天Python相关文章
3. **切换赛道**：开始发目标领域文章
4. **观察数据**：关注阅读量变化
5. **进入流量池**：等待系统推荐

#### 第3步：账号定位
**目标**：明确账号方向

**操作流程**：
1. **选择领域**：根据兴趣选择10大热门赛道
2. **确定定位**：明确目标受众和内容方向
3. **设计IP**：设计个人形象和风格
4. **准备素材**：收集相关素材和案例

#### 第4步：找对标账号
**目标**：学习成功经验

**操作流程**：
1. **搜索关键词**：在公众号搜索相关领域关键词
2. **筛选账号**：选择近半年注册、文章不多但爆款多的账号
3. **关注对标**：关注3-5个对标账号
4. **分析爆款**：拆解对标账号爆款文章
5. **学习经验**：总结对标账号的成功经验

#### 第5步：内容创作
**目标**：创作优质内容

**操作流程**：
1. **选题**：根据对标账号选题
2. **写作**：使用AI写作工作流
3. **优化**：增加个人观点和个人经历
4. **检查**：确保内容质量

#### 第6步：发布
**目标**：选择最佳时间发布

**操作流程**：
1. **选择时间**：根据领域选择最佳时段
2. **选择方式**：单篇发布或多篇合并
3. **设置推荐**：确保平台推荐开关打开
4. **预览检查**：手机端预览
5. **发布**：点击发布

#### 第7步：数据分析
**目标**：了解账号表现

**操作流程**：
1. **查看数据**：阅读量、粉丝增长、互动数据
2. **分析来源**：阅读来源分析
3. **识别异常**：发现异常值文章
4. **总结规律**：总结爆款规律

#### 第8步：优化迭代
**目标**：持续优化账号

**操作流程**：
1. **调整选题**：根据数据调整选题方向
2. **优化内容**：提高内容质量
3. **增加更新**：提高更新频率
4. **建立体系**：形成内容体系
5. **矩阵运营**：考虑多账号矩阵

## 五、常见问题处理流程

### 1. 账号被限流处理流程

```
发现限流 → 检查推荐开关 → 搜一搜检查 → 查看数据 → 分析原因 → 申诉或调整
```

**操作步骤**：
1. **发现限流**：阅读量持续低迷
2. **检查推荐开关**：确认平台推荐开关已打开
3. **搜一搜检查**：搜索公众号名称和文章标题
4. **查看数据**：查看推荐占比
5. **分析原因**：
   - 内容质量问题
   - 标题不吸引人
   - 账号不垂直
   - 敏感词问题
6. **处理方式**：
   - 调整内容策略
   - 优化标题
   - 保持垂直更新
   - 申诉（如有必要）

### 2. 原创标签被滥用处理流程

```
收到通知 → 准备申诉材料 → 撰写申诉 → 提交申诉 → 等待结果 → 调整策略
```

**操作步骤**：
1. **收到通知**：收到滥用原创标签通知
2. **准备材料**：
   - 违规事实说明
   - 违规原因分析
   - 整改措施
   - 承诺书
3. **撰写申诉**：使用申诉模板
4. **提交申诉**：通过公众号后台提交
5. **等待结果**：等待平台审核
6. **调整策略**：
   - 加强规则学习
   - 完善内容审核
   - 强化团队管理

### 3. 流量主开通问题处理流程

```
粉丝不足 → 涨粉方法 → 开通流量主 → 绑定财务信息 → 注意事项
```

**操作步骤**：
1. **粉丝不足**：粉丝未达到500
2. **涨粉方法**：
   - 淘宝买粉（有风险）
   - 微信好友转化
   - 抽奖吸粉
3. **开通流量主**：粉丝达到500后开通
4. **绑定财务信息**：上传身份证、银行卡
5. **注意事项**：
   - 禁止诱导点击
   - 收益每半月结算
   - 注意广告插入规则

### 4. 留言功能问题处理流程

```
无留言功能 → 保持更新 → 等待开通 → 或使用评论区替代
```

**操作步骤**：
1. **无留言功能**：2018年后注册的公众号默认无留言
2. **保持更新**：高频更新、发布原创内容
3. **等待开通**：等待系统随机开通
4. **替代方案**：使用文章评论区替代

### 5. 助推功能使用问题处理流程

```
考虑使用 → 评估风险 → 选择文章 → 使用助推 → 监控结果 → 调整策略
```

**操作步骤**：
1. **考虑使用**：评估是否需要使用助推
2. **评估风险**：了解助推是人工审核，有风险
3. **选择文章**：选择优质文章使用助推
4. **使用助推**：设置助推奖励
5. **监控结果**：关注审核结果和后续流量
6. **调整策略**：
   - 审核通过：继续使用
   - 审核失败：暂停使用，调整内容
   - 流量下降：避免使用

## 六、工作流优化建议

### 1. AI写作工作流优化
- **分步使用**：不要一次性让AI完成所有步骤
- **人工干预**：每个步骤后都进行人工优化
- **建立模板**：根据领域建立自己的提示词模板
- **持续迭代**：根据数据反馈优化提示词

- **二创为主**：以二创为主，避免直接搬运
- **矩阵运营**：多账号矩阵，分散风险
- **数据驱动**：根据数据调整选题和内容

### 3. 文章运营工作流优化
- **固定时间**：固定发布时间，培养用户习惯
- **多篇合并**：入池后多篇合并发布，最大化流量
- **评论引导**：通过评论引导话题，增加互动
- **数据复盘**：每周复盘数据，总结规律

### 4. 账号运营工作流优化
- **矩阵运营**：3-5个账号同时运营，分散风险
- **垂直定位**：保持账号垂直，避免跨领域
- **持续更新**：保持稳定更新频率
- **私域引流**：公域转私域，提高变现效率
