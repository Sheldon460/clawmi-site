## 🎯 核心策略

### 爆款公式（验证有效）
```
爆款 = 热点时效 × 情绪强度 × 传播势能 × 可复制性
```

### 优先级判断（按顺序）
1. **热点时效**：是否在 24 小时黄金期内
2. **情绪强度**：争议/反差/愤怒/惊喜/共鸣
3. **传播势能**：是否具备社交货币属性
4. **可复制性**：能否快速产出多版本测试

## 📈 标题库（已验证有效）

### 类型 1：痛点 + 结果
- "XX 天涨粉 10 万，我只做对了这 3 件事"
- "别再 XX 了，这才是正确姿势"
- "90% 的人都错了，真正有效的方法是..."

### 类型 2：身份 + 反常识
- "作为 XX，我劝你别 XX"
- "XX 从业者不会告诉你的秘密"
- "做了 X 年 XX，我发现..."

### 类型 3：时间承诺 + 明确收益
- "15 分钟搞定 XX，新手也能学会"
- "每天 X 分钟，X 天后见效"
- "X 小时从 0 到 1，完整复盘"

### 类型 4：数字 + 清单
- "X 个 XX 信号，中了 3 个就要警惕"
- "X 款 XX 测评，只推荐这 1 款"
- "X 元预算，这样花最值"

## 🔥 热点快反 SOP（15 分钟出稿）

### Step 1: 热度确认（2min）
```bash
web_search: "[关键词] 热搜/爆款/最新"
→ 确认是否在上升期
→ 记录 Top3 相关话题
```

### Step 2: 样本抓取（3min）
```bash
web_fetch: [爆款链接 1/2/3]
→ 提取标题/封面文案/开头 3 句
→ 记录互动数据（赞/藏/评）
```

### Step 3: 结构拆解（5min）
```
分析维度：
- 标题结构（痛点/身份/数字/反常识）
- 开头钩子（问题/场景/结论/悬念）
- 内容结构（并列/递进/对比/故事）
- 结尾 CTA（互动/关注/转化）
```

### Step 4: 内容产出（3min）
```
产出清单：
- 标题 3-10 条（覆盖不同结构）
- 口播稿 1 条（15-45 秒）
- 封面指令 1 条（给 mi-hua/nano-banana-pro）
- 评论引导语 2-3 条
```

### Step 5: 归档汇报（2min）
```
- 飞书文档创建（markdown → 飞书）
- Obsidian 本地同步
- 群内汇报（结论先行 + 可发布内容）
```
