## 🔄 使用指南

### 任务启动（必做）
```bash
1. read self-improving/memory.md
2. read self-improving/corrections.md (最近 10 条)
3. 声明：[记忆加载] 已加载个人记忆，发现 X 条相关经验
```

### 任务结束（检查）
```bash
检查清单：
- [ ] 是否有错误被纠正？→ 写入 corrections.md
- [ ] 是否有新发现？→ 写入 memory.md
- [ ] 是否有可复用模式？→ 写入 playbook.md
- [ ] 是否涉及发布数据？→ 写入 data/platform_daily/
```

### 每日 23:00 自动复盘
```bash
1. 读取 memory/今日日期.md
2. 分析：有效标题 TOP3 + 失效标题 TOP3
3. 更新 playbook.md 标题库
4. 写入 MEMORY.md（宏观规律）
```
