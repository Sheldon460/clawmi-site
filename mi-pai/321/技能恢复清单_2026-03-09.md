## 🔧 技能 1: xiaohongshu-ops (核心运营)

### 功能描述
小红书运营全流程工具，包含账号定位、选题策划、Viral Copy 爆款复刻、评论管理。

### 安装步骤
```bash
# 1. 克隆技能仓库
git clone https://github.com/sheldon-empire/xiaohongshu-ops.git \
  ~/.openclaw/extensions/feishu-openclaw-plugin/skills/xiaohongshu-ops/

# 2. 验证安装
ls -la ~/.openclaw/extensions/feishu-openclaw-plugin/skills/xiaohongshu-ops/
```

### 核心能力
1. **Step 1 账号定位**: 目标用户画像、价值主张、差异化角度
2. **Step 2 选题策划**: 平台信号抓取、竞品对标分析
3. **Step 3 内容创作**: Viral Copy 爆款复刻、文案生成
4. **Step 6 评论管理**: 个性化回复、风控管理

### 关键文件
- `persona.md` - 人设规范（傲娇嘴硬型）
- `xhs-viral-copy-flow.md` - 爆款复刻流程
- `xhs-comment-ops.md` - 评论管理 SOP

### 使用示例
```bash
# 执行 Viral Copy 爆款复刻
/canghe-xhs-ops viral-copy <爆款笔记 URL> --mode style-only

# 账号定位分析
/canghe-xhs-ops positioning --target "职场效率工具"
```

## 🔧 技能 2: xiaohongshu-mcp (平台工具)

### 功能描述
基于 xpzouying/xiaohongshu-mcp 项目，提供小红书平台 API 能力。

### 安装步骤
```bash
# 1. 安装 MCP Server
mkdir -p ~/.openclaw/skills/xiaohongshu-mcp/
cd ~/.openclaw/skills/xiaohongshu-mcp/

# 2. 下载二进制文件（根据系统选择）
# macOS ARM64:
curl -L https://github.com/xpzouying/xiaohongshu-mcp/releases/latest/download/xiaohongshu-mcp-darwin-arm64 \
  -o xiaohongshu-mcp-darwin-arm64
chmod +x xiaohongshu-mcp-darwin-arm64

# 3. 配置 mcporter
mcporter config add xiaohongshu-mcp \
  --command "./xiaohongshu-mcp-darwin-arm64" \
  --cwd "~/.openclaw/skills/xiaohongshu-mcp/"

# 4. 启动服务
mcporter start xiaohongshu-mcp

# 5. 登录小红书
mcporter call xiaohongshu-mcp.get_login_qrcode
# 使用小红书 App 扫码登录

# 6. 验证登录状态
mcporter call xiaohongshu-mcp.check_login_status
```

### 可用工具
| 工具名称 | 功能 | 必需参数 |
|---------|------|---------|
| `publish_content` | 发布图文 | title, content, images |
| `search_feeds` | 搜索内容 | keyword |
| `get_feed_detail` | 获取笔记详情 | feed_id, xsec_token |
| `like_feed` | 点赞 | feed_id, xsec_token |
| `favorite_feed` | 收藏 | feed_id, xsec_token |
| `post_comment_to_feed` | 发表评论 | feed_id, xsec_token, content |

### 使用示例
```bash
# 发布图文内容
mcporter call xiaohongshu-mcp.publish_content \
  --title "小红书运营工具包上线" \
  --content "今天分享..." \
  --images '["/tmp/cover.jpg", "/tmp/page2.jpg"]' \
  --tags '["小红书运营", "工具分享"]'

# 搜索爆款笔记
mcporter call xiaohongshu-mcp.search_feeds --keyword "效率工具"
```

### 注意事项
- **登录状态**: 大部分功能需要登录后才能使用
- **图片路径**: 支持本地绝对路径或 HTTP/HTTPS URL
- **标题限制**: ≤20 字（严格限制）
- **日发布限制**: ≤50 条

## 🔧 技能 3: canghe-xhs-images (视觉生成)

### 功能描述
小红书封面图批量生成工具，支持多种风格和布局。

### 安装步骤
```bash
# 1. 克隆技能仓库
git clone https://github.com/sheldon-empire/canghe-xhs-images.git \
  ~/.openclaw/extensions/feishu-openclaw-plugin/skills/canghe-xhs-images/

# 2. 安装依赖
cd ~/.openclaw/extensions/feishu-openclaw-plugin/skills/canghe-xhs-images/
npm install

# 3. 配置环境变量
export GOOGLE_GENMEDIA_API_KEY="your-api-key"
```

### 风格选项
| 风格 | 适用场景 |
|------|----------|
| `cute` | 甜美可爱，经典小红书风 |
| `notion` | 简约手绘，知识类内容 |
| `study-notes` | 真实手写笔记风 |
| `bold` | 高冲击力，吸睛型 |
| `minimal` | 极简高级，品牌感 |

### 布局选项
| 布局 | 适用场景 |
|------|----------|
| `sparse` | 极简信息，1-2 个重点 |
| `balanced` | 标准内容，3-4 个要点 |
| `dense` | 高密度知识卡 |
| `list` | 枚举排名类 |
| `comparison` | 对比分析类 |

### 使用示例
```bash
# 生成封面图
/canghe-xhs-images content.md --style notion --layout list

# 批量生成配图
/canghe-xhs-images content.md --style minimal --layout balanced --count 4
```

### 备选方案
- **Nano Banana Pro**: Gemini 图像生成（中文支持更好）
- **即梦 AI**: https://jimeng.jianying.com（手动生成）
- **Canva/稿定设计**: 模板化设计工具
