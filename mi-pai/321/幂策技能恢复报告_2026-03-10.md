## 📊 技能功能详情

### 1. stock-monitor (股票实时监控)

**核心功能**:
- 实时行情监控 (A 股/港股/美股)
- 异动预警 (涨幅>5%、RSI 超买/超卖、成交量异常)
- 板块扫描 (28 个重点板块)
- 飞书推送预警

**预警条件**:
```json
{
  "price_change_pct": 5.0,    // 单日涨幅 > 5%
  "rsi_overbought": 70,       // RSI > 70 超买
  "rsi_oversold": 30,         // RSI < 30 超卖
  "volume_ratio": 3.0,        // 成交量 > 3 倍均值
  "turnover_rate": 10.0       // 换手率 > 10%
}
```

**使用方法**:
```bash
# 启动实时监控
python3 realtime_monitor.py --interval 300

# 加载 watchlist
python3 realtime_monitor.py --watchlist watchlist.json
```

### 2. stock-analysis (股票深度分析)

**核心功能**:
- 8 维评分体系 (基本面/成长性/估值/财务健康/行业地位/管理层/技术面/资金面)
- 估值模型 (PE/PB/PEG/DCF)
- 技术面分析 (均线/MACD/RSI/支撑阻力)
- 持仓管理 (仓位建议/止损止盈)

**评分体系**:
```
90-100 分：⭐⭐⭐⭐⭐ 强烈推荐 (Strong Buy)
80-89 分：  ⭐⭐⭐⭐ 推荐 (Buy)
70-79 分：  ⭐⭐⭐ 持有 (Hold)
60-69 分：  ⭐⭐ 谨慎 (Underperform)
<60 分：    ⭐ 卖出 (Sell)
```
