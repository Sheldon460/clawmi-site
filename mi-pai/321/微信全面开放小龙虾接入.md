## 怎么安装？两步搞定

![安装流程图](/Volumes/My house/Users/Sheldon/.openclaw/workspace/mi-xin/03-内容工厂/3-配图成品区/01_install.png)

**第一步：安装插件**

打开终端，复制这行命令：

```bash
npx -y @tencent-weixin/openclaw-weixin-cli@latest install
```

回车。等待安装完成。

**第二步：扫码登录**

安装完成后，屏幕上会弹出一个二维码。

打开微信，扫一扫。确认登录。

**搞定。**

你的微信和 OpenClaw 已经连上了。

---

## 扫码登录，安全吗？

很多人问：扫码登录，我的微信安全吗？

**放心。**

这个扫码流程，和你在电脑上登录微信网页版是一样的原理。

- 二维码由微信官方生成
- 登录确认需要你在手机上点"确认"
- 不会获取你的密码
- 不会读取你的聊天记录

**你完全掌控。**

## 一些你可能想知道的

**Q：收费吗？**

A：插件本身免费。OpenClaw 的 Token 消耗按正常标准计费。

**Q：支持语音吗？**

A：目前支持文字输入。语音输入还在规划中。

**Q：对话记录会保存吗？**

A：会。和你在其他平台使用 OpenClaw 一样，对话历史自动同步。

**Q：手机没电了怎么办？**

A：换个手机登录微信，继续聊。所有记录都在云端。
