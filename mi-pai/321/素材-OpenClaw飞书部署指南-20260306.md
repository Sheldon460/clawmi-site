# 立省 500！30 分钟把 OpenClaw 在飞书上配到可用

## 📋 核心摘要
这是一份飞书官方出品的 OpenClaw 部署指南，详细介绍了如何在 30 分钟内完成 OpenClaw 在飞书上的配置。文档涵盖本地部署和云端部署（ArkClaw）两种方案，包含完整的权限配置、插件安装、机器人创建等步骤。

## 🎯 关键要点

### 1. OpenClaw 简介
- **定义**: 开源的个人 AI Agent 系统，可运行在个人电脑或服务器上
- **核心价值**: 你说一句话，它就能伸出"钳子"，直接在飞书里帮你把活儿干了

### 2. 为什么选择飞书
| 优势 | 说明 |
|:---|:---|
| 官方插件支持 | 飞书推出 OpenClaw 官方插件，以用户身份调用飞书能力 |
| 中文生态 | 中文界面、文档和客服，更容易上手 |
| 开放能力强 | 能获取更多工作中必要的上下文，玩法更多 |
| 用户基数大 | 国内绝大多数 OpenClaw 用户选择接入飞书 |

### 3. 部署方式对比
| 方式 | 优点 | 缺点 | 适用场景 |
|:---|:---|:---|:---|
| **本地部署** | 数据完全本地，隐私性高；无持续云端费用 | 依赖本地电脑性能；需自行维护；门槛较高 | 个人开发者、对数据安全要求高的团队 |
| **云端部署(ArkClaw)** | 操作简单；24小时在线；性能稳定 | 产生服务器费用；数据存储在云端 | 团队协作、需要7x24小时服务的业务场景 |

### 4. 本地部署关键步骤
1. **环境准备**: 安装 Node.js、Homebrew、Git
2. **安装 OpenClaw**: `npm i -g openclaw`
3. **创建飞书机器人**: 企业自建应用 → 添加机器人能力 → 批量导入权限
4. **安装飞书官方插件**: 
   ```bash
   npm config set registry https://registry.npmjs.org
   curl -o /tmp/feishu-openclaw-plugin-onboard-cli.tgz https://sf3cn.feishucdn.com/obj/open-platformopendoc/90600c5361ebe210fdae13020544bdc7_GmKPCTpnyF.tgz
   npm install /tmp/feishu-openclaw-plugin-onboard-cli.tgz -g
   feishu-plugin-onboard install
   ```
5. **配置 AI 模型**: 获取 API Key（推荐火山引擎、MinMax）
6. **配对与授权**: 发送配对码完成绑定，`/feishu auth` 批量授权

### 5. 云端部署(ArkClaw)关键步骤
1. **开通 ArkClaw**: 火山引擎官网订阅套餐（Lite首月￥9.9，Pro首月￥49.9）
2. **创建飞书应用**: 同本地部署步骤
3. **配置消息渠道**: 在 ArkClaw 后台填入 App ID 和 App Secret
4. **订阅事件与发布**: 配置长链接接收事件，发布应用等待审批
5. **完成配对**: 在飞书中发送消息获取配对码，完成绑定

### 6. 重要风险提示
⚠️ **核心风险**: 插件通过飞书接口连接工作数据，AI 能读到的东西理论上有泄露可能
- **强烈建议**: 现阶段切勿使用公司/企业飞书账号接入！优先使用个人账号体验和测试
- **AI 幻觉**: AI 有时会误解意图或生成不准确的内容
- **不可逆操作**: AI 代发的飞书消息以你的名义发出，发出后即成事实

### 7. 常用指令
| 指令 | 用途 |
|:---|:---|
| `openclaw gateway run` | 启动 AI 网关服务 |
| `openclaw gateway stop` | 停止网关进程 |
| `openclaw gateway restart` | 重启网关 |
| `openclaw doctor` | 自动诊断 |
| `openclaw dashboard` | 启动可视化界面 |
| `/feishu start` | 确认龙虾是否安装成功 |
| `/feishu auth` | 批量完成用户授权 |
| `/feishu doctor` | 检查配置是否正常 |

### 8. 进阶配置
- **流式输出**: `openclaw config set channels.feishu.streaming true`
- **回复模式设置**:
  - 需要@才回复: `openclaw config set channels.feishu.requireMention true --json`
  - 不需要@也回复: `openclaw config set channels.feishu.requireMention false --json`

### 9. 推荐应用场景
- **智能早报生成**: 自动抓取天气、日历、新闻热点和未读邮件摘要
- **邮件与日程管理**: 读取收件箱，提取会议/任务信息生成飞书日程
- **文件智能整理**: 按规则扫描本地文件与云文档，自动分类归档
- **跨端远程办公**: 在手机飞书发送指令，由部署在电脑或服务器的 OpenClaw 执行
- **多 Agent 协作**: 构建专业化分工的 AI 团队（AI 大总管、资讯助理、内容助理、代码助理、任务助理）
