# 短视频脚本模板库

## 🎬 通用脚本结构 (60 秒版本)

### 模板 1: 知识分享类
```
【0-3s】钩子开场
- 抛出痛点/疑问/反差
- 例："90% 的人都做错了！"

【3-15s】问题阐述
- 描述常见误区
- 引发共鸣

【15-45s】解决方案
- 步骤 1/2/3 拆解
- 配合画面演示

【45-55s】总结升华
- 金句总结
- 价值强调

【55-60s】行动号召
- 引导点赞/评论/关注
- 例："学会了吗？评论区告诉我"
```

### 模板 2: 故事叙述类
```
【0-5s】悬念开场
- "我从未想过..."
- "那天发生了一件不可思议的事"

【5-20s】背景铺垫
- 时间/地点/人物
- 建立情境

【20-45s】冲突高潮
- 遇到的问题
- 如何突破

【45-55s】反转结局
- 意外结果
- 情感共鸣

【55-60s】感悟总结
- 从中学到什么
- 引导互动
```

### 模板 3: 产品展示类
```
【0-3s】视觉冲击
- 产品高光镜头
- 效果对比

【3-15s】痛点场景
- 没有产品时的困扰

【15-35s】产品亮相
- 核心功能演示
- 使用过程

【35-50s】效果验证
- 前后对比
- 用户证言

【50-60s】购买引导
- 限时优惠
- 行动指令
```

## 📝 分镜脚本格式

| 镜号 | 时长 | 画面 | 台词/配音 | 音乐/音效 | 备注 |
|:---:|:---:|------|----------|----------|------|
| 1 | 3s | 特写：震惊表情 | "你敢信吗？" | 音效：叮 | 吸引注意 |
| 2 | 5s | 全景：场景展示 | "事情是这样的..." | BGM 起 | 建立情境 |
| 3 | 10s | 中景：操作演示 | "第一步..." | 音效：点击 | 核心内容 |
