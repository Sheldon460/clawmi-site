# 爆款标题库

## 🔥 高点击标题公式

### 反差类 (点击率 +35%)
```
- 以为...结果...
- 本以为...没想到...
- 从...到...的逆袭
- 花 X 万买的教训
- 月薪 3 千到 3 万的差距
```

### 数字类 (收藏率 +28%)
```
- 3 个技巧让你...
- 5 个错误千万别犯
- 7 天搞定...
- 10 年经验总结
- 99% 的人都不知道
```

### 疑问类 (评论率 +42%)
```
- 为什么...?
- 如何...?
- ...是真的吗？
- 你敢信...?
- 到底要不要...?
```

### 经历类 (完播率 +31%)
```
- 我花了 X 天...
- 亲测有效...
- 实测 X 款...
- 历时 X 个月...
- 从 0 开始...
```

### 紧急类 (即时行动)
```
- 最后 X 天...
- 即将截止...
- 限时免费...
- 手慢无...
- 错过等一年...
```

## ⚠️ 标题避坑

### 违规词 (限流风险)
❌ 最、第一、绝对、100%
❌ 赚钱、暴富、躺赚
❌ 医疗功效词
❌ 政治敏感词

### 标题党 (降低信任)
❌ 过度夸张
❌ 文不对题
❌ 低俗擦边

---

## 🎯 A/B 测试建议

每次发布准备 2-3 个标题备选：
1. **保守版**: 直接描述内容
2. **进阶版**: 加入数字/疑问
3. **激进版**: 反差/悬念

观察 24 小时数据，选择最优。
