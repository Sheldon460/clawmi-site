# 幂推 Twitter/X 运营 Playbook (SOP)

> 版本: v1.0 | 创建日期: 2026-03-08 | 状态: 初稿

---

## 1. 运营目标

### 核心 KPI
| 指标 | 目标值 | 监测频率 |
|------|--------|----------|
| 日均发推数 | 3-5 条 | 每日 |
| 互动率 (Engagement Rate) | > 3% | 每周 |
| 粉丝增长率 | > 5%/月 | 每月 |
| 内容转发率 | > 1% | 每周 |

### 内容定位
- **专业领域**: AI、技术、创业、个人成长
- **目标受众**: 技术从业者、创业者、AI 爱好者
- **内容调性**: 专业、深度、有洞察力、偶尔幽默

## 2. 内容策略矩阵

### 2.1 内容类型配比
| 类型 | 占比 | 说明 |
|------|------|------|
| 原创观点 | 40% | 个人思考、行业洞察 |
| 技术分享 | 30% | 教程、工具推荐、代码片段 |
| 互动内容 | 15% | 提问、投票、讨论 |
| 转发评论 | 10% | 优质内容转发+评论 |
| 个人动态 | 5% | 生活、工作日常 |

### 2.2 爆款内容模板

#### 模板 A: 反常识观点
```
【开头】大多数人认为 [常见观点]
【转折】但实际上 [反常识观点]
【论证】原因有三：
1. ...
2. ...
3. ...
【结尾】你的看法是？
```

#### 模板 B: 时间反差
```
【开头】X 年前，我 [当时的状态]
【发展】经历了 [关键转折]
【现状】现在 [现在的状态]
【感悟】最大的收获是 [核心洞察]
```

#### 模板 C: 资源清单
```
【标题】我花了 X 小时整理的 [主题] 资源
【列表】
- 工具 1: ...
- 工具 2: ...
- 工具 3: ...
【福利】完整清单在 [链接]
```

## 3. 发布规范

### 3.1 最佳发布时间 (基于目标受众时区)
| 时段 | 目标地区 | 内容类型 |
|------|----------|----------|
| 08:00-09:00 CST | 亚洲早间 | 晨读/灵感类 |
| 12:00-13:00 CST | 亚洲午休 | 轻松/互动类 |
| 20:00-22:00 CST | 亚洲晚间 | 深度/技术类 |
| 08:00-10:00 EST | 北美早间 | 重要内容 |

### 3.2 推文格式规范

#### 单条推文
- 文字: 200-250 字符最佳
- 换行: 使用空行增加可读性
- 标签: 2-3 个相关标签
- 表情: 适度使用，不超过 3 个

#### Thread (串推)
- 开头推文: 钩子 + 预告
- 中间推文: 每条约 100-150 字符
- 结尾推文: CTA (行动号召)
- 编号: 使用 1/n, 2/n 格式

### 3.3 视觉规范
- 图片尺寸: 1200 x 675 (16:9) 或 1200 x 1200 (1:1)
- 视频时长: 建议 30-90 秒
- 配图风格: 简洁、专业、品牌一致

## 4. 技术操作 SOP

### 4.1 发布流程

#### 标准发推 (文字+图片)
```bash
# 设置环境变量
export X_BROWSER_CHROME_PATH="/Volumes/My house/Applications/Google Chrome.app/Contents/MacOS/Google Chrome"

# 执行发布
cd "/Volumes/My house/Users/Sheldon/.openclaw/skills/canghe-post-to-x/scripts/"
npx -y bun x-browser.ts "推文内容" --image ./图片路径.png
```

#### 视频发布
```bash
npx -y bun x-video.ts "视频描述" --video ./视频路径.mp4
```

#### 长文章发布
```bash
npx -y bun x-article.ts ./文章.md --cover ./封面.jpg
```

### 4.2 发布前检查清单
- [ ] 内容已审核 (无敏感信息)
- [ ] 图片/视频已优化
- [ ] 标签已添加
- [ ] 链接可访问
- [ ] 拼写检查完成

### 4.3 发布后动作
1. 监控前 30 分钟互动情况
2. 回复前 5 条评论
3. 记录数据到追踪表
4. 如有必要，进行跨平台转发

## 5. 互动策略

### 5.1 回复规范
- **速度**: 重要评论 15 分钟内回复
- **态度**: 友好、专业、有价值
- **长度**: 根据情况，简短或详细

### 5.2 主动互动
- 每日浏览关注列表，互动 10-15 条
- 参与热门话题讨论
- 与行业 KOL 建立联系

## 7. 风险管理

### 7.1 内容禁忌
- 政治敏感话题
- 未经证实的传闻
- 攻击性言论
- 版权争议内容

### 7.2 账号安全
- 定期检查登录设备
- 启用 2FA
- 敏感操作二次确认
