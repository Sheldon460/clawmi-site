## 三、节奏控制公式

### 🎵 通用节奏公式：**3-5-8 黄金节拍**

```
0-3秒：钩子（Hook）→ 决定生死
3-8秒：铺垫（Setup）→ 建立信任/背景
8-20秒：展开（Development）→ 核心内容输出
20-35秒：高潮（Climax）→ 情绪峰值/干货爆发
35-45秒：收尾（CTA）→ 行动召唤/互动引导
```

### 🎬 剪辑节奏技术规范

| 元素 | 建议参数 | 说明 |
|-----|---------|-----|
| 镜头时长 | 1.5-3秒/镜 | 快节奏内容可压缩至0.8-1.2秒 |
| 转场方式 | 硬切为主 | 避免花哨特效分散注意力 |
| 字幕出现 | 比人声提前0.2秒 | 制造"阅读领先"效应 |
| 音效点缀 | 每5-8秒一个 | 强化情绪转折点 |
| BGM起伏 | 高潮部分升调/加强鼓点 | 配合内容情绪曲线 |

### 📐 内容密度公式
**有效信息密度 = 核心观点数 / 视频时长(秒)**
- 娱乐类：0.3-0.5个观点/秒
- 知识类：0.5-0.8个观点/秒
- 带货类：0.8-1.2个观点/秒

### ⏱️ 卡点技巧
1. **7秒卡点**：在第7秒设置一个小反转或小高潮，防止第一波流失
2. **15秒卡点**：插入互动提问或选择题，重置观众注意力
3. **结尾前3秒**：快速闪回核心观点，加深记忆
