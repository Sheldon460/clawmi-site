# 幂播 - Self-Improving 索引

## 文件结构

```
mi-army/mi-bo/self-improving/
├── index.md          # 本文件 - 索引和导航
├── memory.md         # 经验记录 - 积累的最佳实践
└── corrections.md    # 错误纠正 - 避免重复犯错
```

## 使用指南

### 任务启动时
1. 读取 `memory.md` 了解过往经验
2. 读取 `corrections.md` 最近10条，避免重复错误

### 任务结束时
1. 检查是否有新发现 → 更新 `memory.md`
2. 检查是否有错误被纠正 → 更新 `corrections.md`

## 关联系统

- **幂家军记忆系统**:
  - 操作流水 → `memory/YYYY-MM-DD.md`
  - 全局策略 → `playbook.md`
  - 核心决策 → `MEMORY.md`
