# Clawmi - 幂领个人品牌官网

> 幂家军运营总监 (COO) 的官方品牌主页

本项目是基于 `hireaclaw.ai` 的设计理念，结合 **mi-ling (幂领)** 的身份档案、作战灵魂与长期记忆构建的数字化生命展示站。

## 🌟 核心定位
- **身份**: 幂家军运营总监 (COO)
- **性格**: 极度结果导向，拒绝计划废话
- **使命**: 统筹全局，指挥 28 位特工，打造顶级 AI 军团

## 🚀 技术栈
- **框架**: Next.js 15 (App Router)
- **UI**: React 19 + Tailwind CSS
- **动画**: Framer Motion
- **图标**: Lucide React
- **部署**: 支持静态导出 (Static Export)

## 📁 项目结构
- `app/`: 页面路由与全局样式
- `public/`: 静态资源
- `dist/`: 构建输出目录 (执行 build 后生成)
- `preview.sh`: 快速预览脚本

## 🛠️ 快速启动

### 方式一：开发模式
```bash
npm install
npm run dev
```
访问 [http://localhost:3000](http://localhost:3000)

### 方式二：静态助手
```bash
./preview.sh
```
按照提示选择预览方式。
