## 二、提示词模板库

### 2.1 视频封面 (16:9, 2K)

**模板结构**: `[风格] + [主体] + [细节] + [光影] + [氛围]`

```
科技感极简主义 + 核心概念可视化 + 光纤数据流 + 侧光轮廓 + 未来感
```

**适用场景**: B 站/YouTube 视频封面、公众号头图

**Imagen 参数**:
- Model: `imagen-4.0-generate-001`
- Aspect Ratio: `16:9`
- Image Size: `2K`

**适用场景**: 小红书笔记封面、Instagram Post

## 三、图像质量检查清单 (QC)

### 3.1 基础合规 ✓

- [ ] 分辨率达标
  - 封面图 ≥ 1920×1080 (16:9)
  - 小红书 ≥ 1080×1440 (3:4)
  - Logo ≥ 1024×1024 (1:1)
- [ ] 文件格式正确 (PNG/WebP)
- [ ] 文件大小合规
  - 公众号 < 5MB
  - 小红书 < 1MB
- [ ] 无版权风险 (字体/素材可商用)

### 3.2 风格一致性 ✓

- [ ] 品牌色使用正确
- [ ] 视觉隐喻匹配内容主题
- [ ] 与历史素材风格连贯

### 3.3 传播优化 ✓

- [ ] 标题文字可读性 (对比度≥4.5:1)
- [ ] 关键信息在安全区内 (60px 边距)
- [ ] 平台适配 (裁剪后核心内容完整)

## 四、成功案例 SOP

### 4.1 封面设计 SOP (基于 GPT-5.4/Netflix AI 等案例)

**Step 1: 哲学提炼** (30 分钟)
1. 读取内容核心思想
2. 撰写 200-300 字设计理念 (`xxx-philosophy.md`)
3. 提取 3-5 个关键视觉元素

**Step 2: 提示词工程** (15 分钟)
1. 从 Nano Banana Pro 库匹配 3-5 个候选提示词
2. 根据设计理念 remix 提示词
3. 附加 sample image 供用户选择

**Step 3: 图像生成** (10 分钟/次)
1. 使用 Google Imagen 生成 (2K, 16:9)
2. 检查输出质量 (使用 QC 清单)
3. 必要时迭代 2-3 次

**Step 4: 交付打包** (5 分钟)
1. 保存至 `mi-hua/output/covers/`
2. 创建飞书文档交付
3. 记录到今日日记

## 五、技术规格标准化

### 5.1 字体系统
- 中文：思源黑体 (Source Han Sans)
- 英文：Inter
- 代码：JetBrains Mono

### 5.2 网格基准
- 基础单元格：8px
- 版式网格：12 列 (移动端) / 24 列 (桌面端)

### 5.3 圆角规范
- 标准圆角：8px
- 大圆角：16px
- 完全圆形：头像/图标

### 5.4 安全边距
- 封面图四边：60px
- 移动端内容区：24px
- 桌面端内容区：48px

### 5.5 最小字号
- 移动端可读性保证：24px
- 桌面端正文：16px

## 七、已验证模式

### 7.1 图像生成流程
```
分析需求 → 选择工具 → 优化提示词 → 生成 → 质量检查 → 交付
```

### 7.2 提示词优化公式
```
风格 + 主体 + 细节 + 光影 + 氛围 = 高质量输出
```

### 7.3 输出格式优先级
```
PNG (质量优先) > WebP (体积优化) > JPG (兼容备用)
```

```
16:9 (视频封面) / 3:4 (小红书) / 1:1 (社交媒体) / 2.35:1 (电影感)
```
