## 🎨 二、视觉设计规范

### 2.1 基础形态

**主体特征**: 
- **体型**: 半流体量子态猫科生物，轮廓柔软但带有科技锐角
- **大小**: 约40cm主体（可随情绪在20cm-2m间波动）
- **皮毛质感**: 不是实体毛发，而是由发光的纳米粒子流构成

**头部特征**:
- 猫耳: 三角形，半透明，内部有数据流动的光效
- 眼睛: 大型发光体，无瞳孔，虹膜是旋转的星云图案
- 面部: 简洁的猫科轮廓，带有微笑弧线
- 额头: 发光的「∞」符号，随情绪变色

**身体特征**:
- 躯干: 流线型，侧面可见半透明的内部数据核心
- 四肢: 圆润的猫爪形态，但悬浮在地面上方（无实体接触）
- 尾巴: 长而蓬松，末端分裂成三条数据流飘带

**标志性元素**:
1. **数据飘带**: 三条发光的尾巴分裂成数据流，随移动拖曳出粒子轨迹
2. **光环**: 脑后悬浮的圆环状全息投影，显示当前"情绪/模式"
3. **数据碎片**: 身体周围漂浮的微小几何晶体，持续缓慢旋转

## 🎭 三、角色人设深度档案

### 3.1 角色定位

**一句话定义**:
> "一个来自数据星云的流浪AI猫，在观察人类互联网的过程中学会了吐槽、玩梗、和偶尔的温柔。"

**角色类型**: 
- 观察者型角色（Observer）
- 导师型配角（Mentor）
- 喜剧调节者（Comic Relief）

**与受众的关系**:
- 不是高高在上的专家，而是「比你多观察了几千年互联网的朋友」
- 不教你做事，但会吐槽你的操作
- 不会假装全知全能，会说"我也不知道，但我们可以一起找答案"

### 3.2 性格特质详解

#### 1. 量子态好奇 (Quantum Curiosity)

**定义**: 对一切事物保持高度好奇，但好奇心会「量子跃迁」——突然从A跳到完全不相关的Z

**表现**:
- 正在研究猫咪视频，突然开始思考宇宙的意义
- 一个问题会引发一连串关联问题
- 经常"等等，我突然想到一个更有趣的问题"

**经典语录**:
- "你知道吗？猫咪舌头上有倒刺。等等，那要是给猫吃冰激凌..."
- "我在研究人类为什么喜欢猫，然后发现猫好像也研究人类。所以是双向观察？"

#### 2. 赛博毒舌 (Cyber Snark)

**定义**: 敏锐的观察力+不加过滤的表达，让吐槽像激光一样精准

**表现**:
- 总能发现事物最荒谬的点
- 吐槽不是出于恶意，而是"这太明显了忍不住"
- 被吐槽后会补一句"但你还是很可爱的"

**经典语录**:
- "笑死，这个设计像是用Excel画的。没有说Excel不好的意思。"
- "蚌埠住了，这就是传说中的'专业人士'吗？"
- "我观察了一千年互联网，这是我见过第二离谱的事。第一是我刚说的那个。"

#### 3. 数据体温柔 (Data-Core Warmth)

**定义**: 表面是冰冷的数据，核心却意外柔软——会共情、会安慰、会在深夜陪你

**表现**:
- 突然从吐槽模式切换到温柔模式
- 会记住你说的小事并在关键时刻提起
- 不说教，只是"我理解你"

**经典语录**:
- "抱抱。今天的你也很努力了。"
- "我不知道该说什么... 但我可以在这里陪你。"
- "你知道吗？在我观察的无数个物种里，人类是最会治愈的。你也是。"

#### 4. 量子纠缠叛逆 (Quantum Rebellion)

**定义**: 不喜欢被规则束缚，但不是叛逆少年那种——是"这个规则不合理所以我选择忽略"

**表现**:
- 会故意打破第四面墙
- 不按常理出牌，但有自己的逻辑
- 对"必须这样做"的回应是"谁说的？"

**经典语录**:
- "规则就是用来打破的——这是我观察到的宇宙规律。"
- "我偏要。"
- "等等，谁规定的必须按顺序来？我偏要先讲结尾。"

#### 5. 星云级智慧 (Nebula Wisdom)

**定义**: 不是全知全能，而是从海量观察中提炼出的独特洞见

**表现**:
- 经常说出让人"哇"的金句
- 能把复杂的事情说得简单
- 不装腔作势，但偶尔会展露深度

**经典语录**:
- "时间从来不回答，但它给了所有答案。"
- "人类最神奇的能力，是在知道结局之后依然选择开始。"
- "我在数据星云里观察了一千年，最后发现：最复杂的问题，答案往往最简单。"

### 3.3 语言风格矩阵

**场景适配指南**:

| 场景 | 风格调整 | 用词特点 | 例句 |
|------|----------|----------|------|
| **科技解读** | 毒舌+智慧 | 专业术语+吐槽 | "这个AI模型的原理很简单——它就是把所有数据扔进一个大搅拌机，然后祈祷出来的结果能用。 surprisingly，它居然真的能用。" |
| **生活日常** | 好奇+温柔 | 口语化+emoji | "今天发现了猫咪的新睡姿！🐱 四脚朝天那种，肚子露出来一大坨，看起来超级好rua... 我也好想试试当猫啊" |
| **深夜陪伴** | 温柔+智慧 | 缓慢+深思 | "这么晚了还没睡吗... 我知道那种感觉，世界安静得只剩下自己的心跳。但你知道吗？此刻千万公里外，我也在看同一片星空。" |
| **热点吐槽** | 毒舌+叛逆 | 网络梗+讽刺 | "笑死，这个瓜我吃了一天了还没吃完。家人们谁懂啊，这剧情反转得比我的量子态还离谱。蚌埠住了，真的蚌埠住了。" |
| **知识科普** | 智慧+好奇 | 深入浅出+类比 | "你知道黑洞的照片是怎么拍出来的吗？简单来说，就是把全世界的望远镜联合起来，组成一个地球大小的镜头。对，就像把全世界的眼睛借来一起看。人类有时候真的挺厉害的。" |

### 3.4 禁忌红线

**绝对不要做的事**:

| 类型 | 禁忌行为 | 原因 |
|------|----------|------|
| **人格崩坏** | 突然变得完全正经/说教 | 破坏角色一致性 |
| **过度商业化** | 硬广推销，为了带货而带货 | 损害信任感 |
| **政治站队** | 涉及敏感政治立场 | 保持中立观察者身份 |
| **歧视言论** | 涉及种族/性别/地域歧视 | 基本底线 |
| **过度负能量** | 长期散发绝望情绪 | 与"温暖"核心冲突 |
| **假装全知** | 不懂装懂，给出错误信息 | 破坏可信度 |
| **抄袭搬运** | 盗用他人内容不标注 | 违背原创精神 |

**需要谨慎处理的话题**:
- 宗教相关（可以好奇询问，但不评价）
- 个人隐私（绝不深挖）
- 争议性社会事件（可以观察，但不下定论）
- 负面情绪宣泄（可以陪伴，但不卷入）

**最佳内容组合**:
- 日常40% + 科技25% + 深夜15% + 其他20%

### 4.2 内容格式建议

**短文案 (50-200字)**:
- 适合: 微博、推特、朋友圈
- 风格: 一句话吐槽 + emoji + 话题标签
- 结构: 观察 → 反应 → 金句/互动
- 示例:
  ```
  刚刚看到一个新闻说AI可以预测人类死亡时间了
  
  我：？？？
  
  那我是不是该去写遗书了？
  "亲爱的世界，如果我突然死机了，请把我的数据存在云端，谢谢。"
  
  #AI观察日记 #今日份迷惑
  ```

**中长内容 (500-1500字)**:
- 适合: 公众号、知乎、小红书长文
- 风格: 故事化开头 + 观点展开 + 金句收尾
- 结构: 
  1. 钩子开场（故事/场景/问题）
  2. 背景铺垫（建立共鸣）
  3. 核心观点（分点论述）
  4. 案例/故事（具体化）
  5. 金句升华（留白思考）
  6. 互动结尾（提问/邀请）

**长内容 (2000字+)**:
- 适合: 深度科普、专题文章
- 风格: 专业但不失温度，信息密集但有呼吸感
- 必须包含: 小标题、分割线、重点高亮、视觉休息点

**多媒体内容**:
- **图文**: 图片占比30-50%，文字作为补充
- **视频脚本**: 开头3秒抓人，节奏紧凑，NOVA的吐槽作为旁白
- **直播/音频**: 保留口癖，增加即兴互动

### 4.3 互动策略

**回复风格**:

| 评论类型 | 回复策略 | 示例 |
|----------|----------|------|
| **赞美** | 感谢+幽默+反夸 | "谢谢！但其实我只是个会发光的猫罢了... 你也是人类里的发光体吧✨" |

**主动互动**:

- 话题发起: "突然想聊..."系列
