## 龙虾是什么？

OpenClaw，因为Logo是一只卡通龙虾，被网友们亲切地称为"龙虾"。

它是奥地利程序员 Peter Steinberger 去年11月推出的开源AI智能体框架。短短几个月，GitHub Star数已经突破 **27万** ，成为开源软件史上增长最快的项目之一。

简单说，OpenClaw是一只"有手有脚"的AI。

以前的AI，你问它问题，它给你答案。但答案怎么用，还得你自己来。

OpenClaw不一样。它能直接操作你的电脑：

- 打开浏览器帮你搜航班
- 整理桌面文件按日期分类
- 批量改PPT、发邮件
- 盯着股市大盘
- 写代码、测bug

它不只是回答，而是执行。

就像你雇了一名24小时待命的数字员工，只需要告诉它要做什么，剩下的它自己搞定。

## 普通人如何上手？

如果你也想养一只自己的龙虾， here's how：

### 第一步：准备环境

OpenClaw支持两种部署方式：

- **本地部署** ：需要一台性能还不错的电脑（推荐Mac或Linux）
- **云服务器部署** ：腾讯云Lighthouse、阿里云ECS等都可以

对于新手，**云服务器更友好** ——不需要折腾本地环境，随时随地可以访问。

### 第二步：安装OpenClaw

如果你选择腾讯云Lighthouse，官方已经提供了一键安装镜像。创建实例时选择OpenClaw镜像，几分钟就能跑起来。

如果是其他环境，官方文档提供了详细的安装指南：

```
# 克隆仓库
git clone https://github.com/openclaw/openclaw.git

# 进入目录
cd openclaw

# 安装依赖
npm install

# 启动服务
npm start
```

### 第三步：配置模型

OpenClaw本身只是一个框架，需要接入大模型才能工作。

目前支持的模型包括：

- OpenAI GPT-4/GPT-3.5
- 智谱GLM-4
- 月之暗面Kimi
- 阿里云Qwen
- 字节豆包

在配置文件中填入你的API Key即可。

### 第四步：添加Skills

Skills是OpenClaw的能力扩展，相当于给龙虾装上了不同的"工具手"。

官方Skills市场提供了丰富的选择：

- **Browser** ：浏览器自动化
- **FileSystem** ：文件系统操作
- **Code** ：代码编写与执行
- **Slack/Discord/Telegram** ：IM渠道集成
- **GitHub** ：代码仓库管理

安装Skill很简单：

```
# 安装browser skill
openclaw skill install browser

# 安装filesystem skill
openclaw skill install filesystem
```

### 第五步：开始对话

一切准备就绪后，你就可以通过聊天界面或命令行与OpenClaw对话了。

试试这些指令：

- "帮我搜索明天北京到上海的航班，整理成表格"
- "把桌面上的PDF文件按日期分类整理"
- "帮我写一段Python代码，实现图片批量压缩"
