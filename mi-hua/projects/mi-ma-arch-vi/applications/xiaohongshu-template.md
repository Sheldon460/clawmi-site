# 小红书封面模板规范

## 竖版封面 (3:4) - 主力模板

### 基础参数
- **尺寸**: 1080 × 1440 px
- **分辨率**: 72 dpi (网络) / 300 dpi (印刷)
- **色彩模式**: RGB
- **安全边距**: 60px (四边)

### 版式结构

```
┌─────────────────────────────┐  ↑
│      [顶部安全区 60px]       │  │
│                             │  │
│    ┌─────────────────┐      │  │
│    │                 │      │  │
│    │   主视觉区域     │      │  │ 780px
│    │   (图片/图形)   │      │  │
│    │                 │      │  │
│    └─────────────────┘      │  │
│                             │  │
│    ─────────────────────    │  │
│                             │  │
│    主标题 (H2, Bold)        │  │
│    副标题 (H4, Regular)     │  │ 360px
│                             │  │
│    [标签/话题 # ]           │  │
│                             │  │
│      [底部安全区 60px]      │  ↓
└─────────────────────────────┘
```

### 色彩应用

#### 深色主题 (推荐主版)
- **背景**: #0A1628 或渐变 `#0A1628 → #1E293B`
- **主标题**: #FFFFFF
- **副标题/正文**: #94A3B8
- **高亮/强调**: #00D4AA (电路青)
- **标签**: #6366F1 (数据紫)

#### 浅色主题
- **背景**: #FFFFFF 或 #F8FAFC
- **主标题**: #0A1628
- **副标题/正文**: #475569
- **高亮/强调**: #00D4AA
- **标签**: #6366F1

### 字体规范

| 元素 | 字体 | 字号 | 字重 | 颜色 |
|------|------|------|------|------|
| 主标题 | 思源黑体 | 48px | Bold | 白/黑 |
| 副标题 | 思源黑体 | 28px | Medium | 灰 |
| 正文 | 思源黑体 | 24px | Regular | 灰 |
| 标签 | Inter | 20px | Medium | 强调色 |

### 图形元素

1. **Logo位置**: 右上角或左上角，宽度 80-100px
2. **装饰线条**: 电路青 (#00D4AA) 1-2px 线条
3. **几何图形**: 45度角方块、电路节点
4. **发光效果**: 主视觉区周围添加青色光晕

## 横版封面 (16:9) - 视频封面/B站

### 基础参数
- **尺寸**: 1920 × 1080 px
- **分辨率**: 72 dpi
- **色彩模式**: RGB

### 版式结构

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│   ┌─────────────┐                                       │
│   │             │    主标题 (H1)                        │
│   │  视觉区域   │    ───────────────────────              │
│   │  (右对齐)   │    副标题 + 描述                        │
│   │             │                                       │
│   └─────────────┘    [标签] [Logo]                        │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 横版特有规范

- **主标题**: 72-96px，Bold
- **副标题**: 36-48px，Medium
- **视觉区**: 占右侧40%，可出血
- **Logo**: 左下角或右下角，120px

---

## 方形封面 (1:1) - 头像/头像框

### 基础参数
- **尺寸**: 1080 × 1080 px
- **用途**: 微信头像、知乎头像、社交媒体头像

### 设计要点

1. **中心聚焦**: Logo图形置于中心
2. **安全区**: 中心80%区域为安全区
3. **圆形裁切**: 考虑平台自动裁切成圆形的情况
4. **小尺寸测试**: 确保64×64px下仍可识别
