# 网站界面风格指南

## 整体风格

### 设计语言
- **风格**: 赛博朋克 + 极客美学 + 建筑精确感
- **基调**: 深色主题为主，浅色主题为辅
- **氛围**: 专业、科技感、可信赖、未来感

### 核心特征
1. **深色主导**: #0A1628 作为主背景
2. **电光青高亮**: #00D4AA 用于交互和重点
3. **几何结构**: 45度角、网格系统
4. **微光效果**: 柔和的发光和阴影

---

## 色彩应用

### 深色模式 (默认)

| 元素 | 颜色 | 用途 |
|------|------|------|
| **背景** | #0A1628 | 主页面背景 |
| **卡片** | #1E293B | 卡片、面板背景 |
| **边框** | #334155 | 分割线、边框 |
| **主要文字** | #F1F5F9 | 标题、重要文字 |
| **次要文字** | #94A3B8 | 正文、描述 |
| **禁用** | #475569 | 禁用状态文字 |
| **主交互** | #00D4AA | 按钮、链接、高亮 |
| **次交互** | #6366F1 | 次级按钮、标签 |
