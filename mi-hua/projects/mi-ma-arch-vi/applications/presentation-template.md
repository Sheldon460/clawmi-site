# 演示文稿模板规范

## 整体风格

### 设计基调
- **风格**: 赛博朋克 × 工程美学
- **主题**: 深色为主，强调科技感和专业性
- **比例**: 16:9 (标准) 或 4:3 (备选)

### 色彩应用

#### 深色主题 (推荐)
| 元素 | 颜色 | 用途 |
|------|------|------|
| **背景** | #0A1628 | 主背景 |
| **卡片/区块** | #1E293B | 内容卡片 |
| **主标题** | #FFFFFF | 大标题 |
| **副标题** | #E2E8F0 | 次级标题 |
| **正文** | #94A3B8 | 正文内容 |
| **强调** | #00D4AA | 关键词、数据 |
| **次要强调** | #6366F1 | 标签、注释 |

## 页面模板

### 1. 封面页 (Cover)

#### 布局
```
┌─────────────────────────────────────────────────┐
│                                                 │
│     ┌─────────────────────────────────────┐    │
│     │                                     │    │
│     │   ◇  LOGO  ◇                        │    │
│     │                                     │    │
│     │   ───────────────────────           │    │
│     │                                     │    │
│     │   主标题 (H1, 60px, Bold)           │    │
│     │   副标题 (H3, 32px, Regular)        │    │
│     │                                     │    │
│     │   演讲者：幂码-架构                   │    │
│     │   日期：2026年X月X日                  │    │
│     │                                     │    │
│     └─────────────────────────────────────┘    │
│                                                 │
│   [底部装饰：电路图案/网格线]                     │
│                                                 │
└─────────────────────────────────────────────────┘
```

#### 设计规范
- **背景**: 深色渐变 `linear-gradient(135deg, #0A1628 0%, #1E293B 100%)`
- **Logo**: 尺寸 120×120px，位于顶部中央
- **分隔线**: 2px，电光青，宽度 200px
- **主标题**: 60px，Bold，白色
- **副标题**: 32px，Regular，浅灰 #E2E8F0
- **装饰**: 底部电路纹理，透明度 10%

#### 数据可视化配色

**柱状图/条形图配色**:
- 主色: #00D4AA (电路青)
- 辅助色1: #6366F1 (数据紫)
- 辅助色2: #EC4899 (调试粉)
- 辅助色3: #F97316 (警报橙)
- 背景: #1E293B
- 网格线: #334155 (30%透明度)

**折线图配色**:
- 主线: #00D4AA，线宽3px
- 辅助线: #6366F1，线宽2px
- 数据点: 白色填充，边框#00D4AA
- 填充区域: #00D4AA 20%透明度

**饼图/环形图配色**:
```
扇区1: #00D4AA (电路青)   - 主要数据
扇区2: #6366F1 (数据紫)   - 次要数据
扇区3: #EC4899 (调试粉)   - 第三数据
扇区4: #F97316 (警报橙)   - 第四数据
扇区5: #22C55E (成功绿)   - 第五数据
其他:  #475569 (深灰)      - 其他
```

## 通用组件规范

### 按钮样式

**主要按钮 (Primary)**
```
背景: #00D4AA
文字: #0A1628 (深色)
字体: 思源黑体 Medium 16px
圆角: 8px
高度: 48px
内边距: 0 32px
悬停: 背景变亮 10%，添加阴影
点击: 背景变暗 10%
```

**次要按钮 (Secondary)**
```
背景: transparent
边框: 2px solid #00D4AA
文字: #00D4AA
字体: 思源黑体 Medium 16px
圆角: 8px
高度: 48px
悬停: 背景 #00D4AA 10%透明度
```

**幽灵按钮 (Ghost)**
```
背景: transparent
文字: #94A3B8
字体: 思源黑体 Regular 14px
悬停: 文字变白
```

### 卡片组件

**标准卡片**
```
背景: #1E293B
圆角: 16px
内边距: 24px
边框: 1px solid #334155 (可选)
阴影: 0 4px 6px rgba(0, 0, 0, 0.3)
```

**悬停效果**
```
边框: 1px solid #00D4AA
阴影: 0 0 20px rgba(0, 212, 170, 0.2)
变换: translateY(-4px)
```

### 标签/徽章

**主标签**
```
背景: #00D4AA 20%透明度
文字: #00D4AA
字体: 12px Medium
圆角: 4px
内边距: 4px 8px
```

**次标签**
```
背景: #6366F1 20%透明度
文字: #6366F1
```
