## 2. Logo系统

### 2.1 标志设计概念

**图形标志（Symbol）**
核心图形由三个元素融合构成：
1. **建筑结构线** - 代表架构师的精确与层级
2. **代码括号** `{ }` - 代表编程与数字世界
3. **电路节点** - 代表连接与系统性思维

**设计手法**:
- 使用单线等宽线条构建，体现工程师的精确
- 45度角切割增加动感与现代感
- 负空间设计确保小尺寸识别度

### 2.2 Logo版本

#### A. 主Logo（图形+文字组合）
- 横向组合：图形在左，文字在右
- 比例：图形高度 = 文字cap高度 × 1.2
- 间距：图形与文字间距 = 图形宽度 × 0.25

#### B. 图形标志（独立使用）
- 最小使用尺寸：数字端16px，印刷端5mm
- 安全空间：标志高度的25%

#### C. 文字标志
- 英文：MI-MA-ARCH (等宽字体，全大写)
- 中文：幂码-架构

#### D. 反白版本
- 深色背景使用反白Logo
- 主色渐变背景使用纯白Logo

## 5. 视觉元素

### 5.1 Icon系统

**线性Icon (Line Icons)**
- 描边宽度: 1.5px
- 端点样式: 圆角
- 转角样式: 圆角
- 尺寸规范: 16px, 20px, 24px, 32px, 48px

**填充Icon (Filled Icons)**
- 实心填充
- 尺寸规范: 16px, 20px, 24px, 32px, 48px
- 小尺寸增加细节处理

**风格特征**:
- 几何化设计
- 2px圆角处理
- 45度角元素呼应Logo
- 1.5px统一描边

### 5.2 图形装饰元素

**网格系统**
- 基础网格: 8px单元格
- 版式网格: 12列/24列
- 边距比例: 8px, 16px, 24px, 32px, 48px, 64px

**几何图形元素**
1. **斜切方块** - 45度旋转方块，代表建筑结构
2. **电路节点** - 圆点+连接线，代表系统思维
3. **代码符号** - `{ }` 抽象化处理
4. **网格背景** - 细密网格，科技感背景

**动态元素**
- 发光效果 (Glow)
- 扫描线 (Scanline)
- 故障艺术 (Glitch) - 谨慎使用

### 5.3 线条系统

**线条类型**
1. **实线** - 主要分隔线，1px
2. **虚线** - 次要分隔线，4px线+4px间隔
3. **点线** - 装饰线，1px点+3px间隔

**线条粗细**
- 细线: 0.5px
- 标准: 1px
- 粗线: 2px
- 特粗: 4px

### 5.4 光影效果规范

**发光效果 (Glow)**
```css
box-shadow: 0 0 20px rgba(0, 212, 170, 0.5);
/* 主色发光 */

box-shadow: 0 0 30px rgba(99, 102, 241, 0.4);
/* 辅助色发光 */
```

**渐变光晕**
```css
background: radial-gradient(circle at 30% 30%, 
  rgba(0, 212, 170, 0.3) 0%, 
  transparent 70%);
```

**扫描线效果**
```css
background: repeating-linear-gradient(
  0deg,
  transparent,
  transparent 2px,
  rgba(0, 212, 170, 0.03) 2px,
  rgba(0, 212, 170, 0.03) 4px
);
```

## 6. 应用场景模板

### 6.1 小红书封面模板

**竖版封面 (3:4)**
- 尺寸: 1080×1440px
- 布局: 顶部Logo，中间主视觉，底部标题
- 安全区: 四边各留60px
- 字体层级: 主标题H2，副标题H4
- 推荐配色: 深色背景+电路青高亮

**横版封面 (16:9)**
- 尺寸: 1920×1080px
- 适用: 视频封面、B站
- 布局: 左侧文字区，右侧视觉区
- 字体层级: 主标题H1，副标题H3

### 6.2 网站界面风格

**整体风格**
- 深色主题为主，浅色主题为辅
- 圆角: 8px标准，16px大圆角，完全圆形头像
- 阴影: 多层阴影增加层次
- 间距: 8px为基准的倍数系统

**组件规范**
- 按钮: 高度40/48/56px，圆角8px
- 输入框: 高度40/48px，边框1px
- 卡片: 圆角12/16px，内边距16/24/32px
- 导航栏: 高度64/72px

**响应式断点**
- 移动端: < 768px
- 平板: 768px - 1024px
- 桌面: 1024px - 1440px
- 大屏: > 1440px

### 6.3 名片设计

**标准名片 (90×54mm)**
- 材质建议: 哑光黑卡纸+烫电光青
- 正面: Logo+姓名+职位+联系方式
- 背面: 品牌口号或二维码
- 字体: 中文思源黑体，英文Inter

**数字名片**
- 尺寸: 1200×630px (社交分享图)
- 适用: 微信头像、LinkedIn封面

### 6.4 社交媒体头像

**头像规范**
- 尺寸: 400×400px (最小)
- 设计: 图形标志居中
- 背景: 主色或渐变背景
- 各平台适配: 圆形裁切

### 6.5 演示文稿模板

**PPT/Keynote规范**

**封面页**
- 背景: 深色渐变+网格底纹
- 标题: H1层级，电光青高亮
- 副标题: H3层级，中灰色
- 装饰: 角落几何元素

**内容页**
- 背景: 纯白或极浅灰
- 标题区: 顶部，H2层级
- 内容区: 左侧文字，右侧视觉
- 页码: 底部居中或右对齐

**数据页**
- 图表配色: 主色+辅助色渐变
- 重点数据: 电光青高亮
- 背景: 深色卡片式

**结尾页**
- 品牌口号
- 联系方式
- 二维码
- 简洁Logo

## 8. 使用规范

### 8.1 Logo使用禁区

1. **禁止拉伸变形** - 保持原始比例
2. **禁止改变颜色** - 使用标准色值
3. **禁止添加效果** - 投影、描边、渐变需按规范
4. **禁止旋转倾斜** - 保持水平垂直
5. **禁止复杂背景** - 确保Logo识别度

### 8.2 色彩使用比例

- **60%** - 主色/中性色（背景、大面积）
- **30%** - 辅助色（卡片、区块）
- **10%** - 强调色（按钮、重点、CTA）

### 8.3 字体使用层级

- **标题**: 思源黑体 Bold / Inter Bold
- **正文**: 思源黑体 Regular / Inter Regular
- **代码**: JetBrains Mono / Source Code Pro
- **展示**: Space Grotesk (Logo/装饰)
