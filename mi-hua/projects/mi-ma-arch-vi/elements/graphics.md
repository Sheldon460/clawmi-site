# 图形装饰元素规范

## 设计原则

### 视觉语言
图形装饰元素遵循"数字建筑"美学，体现以下特征：
- **几何精确**: 基于数学比例的几何形状
- **结构层次**: 体现架构师的分层思维
- **电路隐喻**: 融入电路板走线元素
- **赛博质感**: 发光、渐变、透明效果

### 核心图形

#### 1. 六边形网格 (Hexagon Grid)
```
    ____
   /    \
  /      \
  \      /
   \____/
```
- **用途**: 背景纹理、装饰边框
- **参数**: 边长 20-50px，间距 2-4px
- **颜色**: 10-20% 透明度白色或青色
- **应用**: 科技背景、数据可视化底纹

#### 2. 电路节点 (Circuit Nodes)
```
    ●─────●
    │     │
    ●─────●
```
- **用途**: 连接线、流程图、装饰线条
- **参数**: 节点直径 4-8px，线宽 1-2px
- **颜色**: 电光青 #00D4AA 或数据紫 #6366F1
- **应用**: 分隔线、信息流向、装饰

#### 3. 斜切方块 (Chamfered Square)
```
  ╱‾‾‾‾‾‾╲
  │      │
  │      │
  ╲______╱
```
- **用途**: 按钮、卡片、标签容器
- **参数**: 切角 45度，深度 8-12px
- **颜色**: 渐变填充或半透明
- **应用**: CTAButton、特性卡片、标签

#### 4. 代码符号 (Code Brackets)
```
  { code }
  < tag />
  [ data ]
```
- **用途**: 装饰性文字、代码展示
- **参数**: 括号 2-3px 描边，代码文字等宽
- **颜色**: 括号电光青，文字白色/灰色
- **应用**: 技术文章、代码片段展示

#### 5. 扫描线 (Scanlines)
```
  ─────────
  ─────────
  ─────────
```
- **用途**: 复古科技感背景
- **参数**: 线高 1px，间距 3-4px
- **颜色**: 5-10% 透明度白色
- **应用**: 视频封面、复古风格设计

## 图形组合规范

### 角部装饰 (Corner Decorations)

#### 标准角标
```
  ┌──────
  │
  │
```
- **尺寸**: 30-50px
- **线条**: 2px，电光青
- **端点**: 圆点或箭头

#### 电路角
```
  ●────
  │
  │
```
- **节点**: 6px 圆形
- **连线**: 1.5px
- **颜色**: 电光青渐变

### 边框装饰 (Border Decorations)

#### 科技边框
```
  ╔═══════════════════╗
  ║                   ║
  ╚═══════════════════╝
```
- **外框**: 2px 实线
- **四角**: 斜切或装饰节点
- **颜色**: 电光青或白色

#### 虚线边框
```
  ┌─ ─ ─ ─ ─ ─ ─ ─ ─ ─┐
  │                   │
  └─ ─ ─ ─ ─ ─ ─ ─ ─ ─┘
```
- **线型**: 4px线 + 4px间隔
- **用途**: 辅助区域、提示框

### 背景装饰 (Background Decorations)

#### 网格背景
```
  ┼────┼────┼────┼
  │    │    │    │
  ┼────┼────┼────┼
```
- **网格**: 20-40px
- **线条**: 0.5px
- **透明度**: 5-10%

#### 渐变光晕
```
  ○ 扩散渐变
```
- **中心**: 电光青，30%透明度
- **边缘**: 完全透明
- **尺寸**: 覆盖50-80%区域

## 应用场景示例

### 网站Hero区域
```
[渐变背景 #0A1628 → #1E293B]
    + [网格纹理 5%透明度]
    + [右上角电路节点装饰]
    + [左下角斜切方块装饰]
    + [中央内容区域]
```

### 卡片组件
```
[卡片背景 #1E293B]
    + [顶部边框 2px 电光青渐变]
    + [左上角/右下角 角标装饰]
    + [内容区域]
```

### 按钮样式
```
[斜切方块按钮]
    + [渐变背景 #00D4AA → #00B894]
    + [内部发光效果]
    + [悬停: 外发光 #00D4AA 30%]
```

## 资源文件

### 提供的装饰元素

| 文件名 | 格式 | 描述 |
|--------|------|------|
| hexagon-grid.svg | SVG | 六边形网格图案 |
| circuit-lines.svg | SVG | 电路连接线 |
| corner-brackets.svg | SVG | 角部括号装饰 |
| scanlines.svg | SVG | 扫描线纹理 |
| gradient-glows.svg | SVG | 渐变光晕效果 |

### 使用方式

```html
<!-- 作为背景图案 -->
<div style="background-image: url('hexagon-grid.svg'); opacity: 0.1;">

<!-- 作为装饰元素 -->
<img src="corner-brackets.svg" style="position: absolute; top: 0; right: 0;">

<!-- 作为分割线 -->
<div style="background: url('circuit-lines.svg') repeat-x;">
```
