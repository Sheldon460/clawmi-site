# Icon系统规范

## 设计原则

### 核心理念
Icon设计遵循"数字建筑"理念，融合以下元素：
- **几何化**: 基础几何形状组合
- **等宽线条**: 1.5px统一描边
- **圆角处理**: 2px圆角，柔和科技感
- **45度角**: 呼应Logo设计元素

### 网格系统
- **基础网格**: 24×24px
- **安全区域**: 2px边距
- **线条位置**: 基于1px网格对齐

## Icon风格

### 线性Icon (Line Icons)
- **描边宽度**: 1.5px
- **端点样式**: 圆角 (Round Cap)
- **转角样式**: 圆角 (Round Join)
- **尺寸规格**: 16px, 20px, 24px, 32px, 48px

**使用场景**:
- 导航菜单
- 列表项标记
- 表单输入提示
- 工具栏按钮

### 填充Icon (Filled Icons)
- **填充方式**: 实心填充
- **尺寸规格**: 16px, 20px, 24px, 32px, 48px
- **小尺寸处理**: 16px/20px增加细节

**使用场景**:
- 状态指示
- 功能开关
- 选中标记
- 重要操作按钮

## Icon使用规范

### 尺寸使用

| 场景 | 尺寸 | 说明 |
|------|------|------|
| 导航栏图标 | 24px | 标准导航尺寸 |
| 工具栏图标 | 20px | 紧凑工具栏 |
| 列表图标 | 16px | 列表项标记 |
| 按钮图标 | 16-20px | 按钮内图标 |
| 大图标/空状态 | 48-64px | 强调展示 |
| 超大图标/品牌 | 96-128px | 品牌展示 |

### 间距规范

```
图标与文字间距: 8px (标准) / 4px (紧凑) / 12px (宽松)
图标组间距: 16px (标准) / 8px (紧凑) / 24px (宽松)
图标与边框间距: 12px (标准)
```

### 颜色使用

**默认状态**:
- 浅色背景: #475569 (深灰)
- 深色背景: #94A3B8 (中灰)

**激活/悬停状态**:
- 主色: #00D4AA (电路青)
- 次色: #6366F1 (数据紫)

**状态颜色**:
- 成功: #22C55E
- 警告: #EAB308
- 错误: #EF4444
- 信息: #3B82F6

### 可访问性

**对比度要求**:
- 图标与背景: 对比度 ≥ 3:1
- 文字与背景: 对比度 ≥ 4.5:1

**尺寸要求**:
- 可点击图标最小尺寸: 44×44px
- 推荐触摸目标: 48×48px

### 自定义Icon

如需自定义Icon，请遵循以下规范：
1. 使用24×24px画布
2. 1.5px描边宽度
3. 2px圆角处理
4. 基于几何形状构建
5. 保持视觉重量一致
