### 2. 视频处理类

#### 2.1 视频帧提取
- **技能名称**: `video-frames`
- **状态**: ✅ 可用
- **依赖**: `ffmpeg` (brew install ffmpeg)
- **功能**: 
  - 提取视频关键帧
  - 生成视频封面/缩略图
  - 按时间戳截取画面
- **命令**:
```bash
# 提取第一帧
{baseDir}/scripts/frame.sh /path/to/video.mp4 --out /tmp/frame.jpg

# 提取指定时间点帧
{baseDir}/scripts/frame.sh /path/to/video.mp4 --time 00:00:10 --out /tmp/frame-10s.jpg
```

#### 2.2 视频剪辑 (videocut)
- **技能名称**: `videocut:剪辑`
- **状态**: ✅ 可用
- **功能**:
  - 口播视频剪辑
  - 去除口误/停顿/静音
  - 自动节奏优化
- **流程**: FunASR 转录 → 口误识别 → 用户确认 → FFmpeg 剪辑

#### 2.3 程序化视频 (Remotion)
- **技能名称**: `remotion-best-practices`
- **状态**: ✅ 可用
- **功能**: React 为基础的视频编程生成
- **最佳用途**: 数据可视化视频、动态图表、模板化视频

## 📚 核心 SOP 与最佳实践

### SOP 1: 口播视频制作流程
```
1. 脚本准备 (mi-wen) → 2. 录音 → 3. FunASR 转录 
→ 4. 口误识别 → 5. 用户确认 → 6. videocut 剪辑 
→ 7. Whisper 字幕生成 → 8. 字幕烧录 → 9. 输出
```

### SOP 2: 漫剧制作流程
```
1. 小说/脚本 (mi-wen) → 2. 分镜图生成 (mi-hua/nano-banana-pro) 
→ 3. 配音 (mi-sheng/sherpa-onnx-tts) → 4. 视频合成 (seedance-video/comic-drama-maker) 
→ 5. 字幕添加 → 6. 输出
```

### SOP 3: 视频分析流程
```
1. 收集竞品视频 URL → 2. summarize 提取内容 
→ 3. video-frames 提取关键帧 → 4. 分析脚本结构/节奏 
→ 5. 输出分析报告 (skill-feishu-docx-powerwrite)
```

### SOP 4: 双重产出协议

凡是正式文档必须：
