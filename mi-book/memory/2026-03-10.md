## 💡 经验总结

### 验证有效的模式

#### 1. 技能安装流程
```bash
# 克隆仓库
git clone --depth 1 https://github.com/xpzouying/xiaohongshu-mcp.git \
  ~/.openclaw/extensions/feishu-openclaw-plugin/skills/xiaohongshu-mcp/

# 构建二进制
cd ~/.openclaw/extensions/feishu-openclaw-plugin/skills/xiaohongshu-mcp/
go build -o xiaohongshu-mcp-server .

# 创建符号链接
ln -sf xiaohongshu-mcp-server xiaohongshu-mcp-darwin-arm64

# 配置 mcporter (添加 cwd)
# 启动服务
./xiaohongshu-mcp-server --port ":18061"
```

#### 2. HTTP API 调用格式
```bash
# 搜索笔记
curl -X POST http://localhost:18061/api/v1/feeds/search \
  -H "Content-Type: application/json" \
  -d '{"keyword":"效率工具"}'

# 获取详情
curl -X POST http://localhost:18061/api/v1/feeds/detail \
  -H "Content-Type: application/json" \
  -d '{"feed_id":"xxx","xsec_token":"xxx"}'

# 发布笔记
curl -X POST http://localhost:18061/api/v1/publish \
  -H "Content-Type: application/json" \
  -d '{
    "title": "标题",
    "content": "正文内容",
    "images": ["/path/to/image.jpg"]
  }'
```

#### 3. 内容创作模板
```
标题：🔥 [年份] 职场人必看｜[数量] 个 [核心效果]
正文结构:
1. 引导语（共鸣 + 价值主张）
2. 工具清单（编号 + 功能点 + 提示）
3. 使用心得（总结 + 建议）
4. 互动提问（评论区引导）
5. 话题标签（9 个左右）
```
