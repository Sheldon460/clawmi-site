## 二、封面设计CTR优化清单

### 2.1 爆款封面黄金公式
```
点击率 = 视觉冲击 × 信息密度 × 情绪共鸣
```

### 2.2 封面设计要素检查表

| ✅ 必做项 | 执行标准 | 示例 |
|-----------|---------|------|
| **主标题** | 字号≥60pt，占据画面上半区1/3 | "AI搞钱" / "效率神器" |
| **副标题** | 补充具体利益点，≤10字 | "月入过万"/"省2小时" |
| **数字锚点** | 阿拉伯数字优先，放大处理 | "3个"/"5步"/"99%" |
| **对比元素** | Before/After或人vs AI效果 | 左右分屏展示 |
| **人物出镜** | 真人表情增强信任感 | 惊讶/惊喜表情最佳 |
| **品牌标识** | 统一视觉锤，强化账号记忆 | 固定位置logo/水印 |

### 2.3 配色方案推荐
| 场景 | 主色调 | 辅助色 | 适用内容 |
|------|--------|--------|----------|
| 效率工具 | #FF6B6B (活力红) | #FFE66D (亮黄) | 时间管理、自动化 |
| 搞钱副业 | #FFD93D (金黄) | #6BCB77 (财富绿) | 变现攻略、案例分享 |
| 入门教程 | #4D96FF (科技蓝) | #A8D8EA (浅蓝) | 新手友好、零基础 |
| 避坑指南 | #FF9F45 (警示橙) | #FC5185 (强调粉) | 常见错误、注意事项 |

### 2.4 CTR提升技巧
- **首图测试**: A/B测试3版封面，24小时内选CTR最高者
- **动态封面**: 视频笔记前3秒必须出现核心利益点
- **文字占比**: 封面文字面积控制在30%-40%
- **留白原则**: 边缘留足安全区，避免被UI遮挡

## 三、种草文案公式

### 3.1 万能文案结构：SCQA-R模型
```
S - Situation (情境): 描述读者当前困境
C - Complication (冲突): 指出问题严重性
Q - Question (疑问): 提出解决方案预告
A - Answer (答案): 给出具体方法/工具
R - Result (结果): 展示效果/号召行动
```

### 3.2 开头钩子模板（3秒留住用户）
| 类型 | 模板句式 | 适用场景 |
|------|---------|----------|
| **痛点直击** | "还在手动XXX？我发现了更聪明的办法..." | 效率工具类 |
| **数字冲击** | "用了这个AI工具，我的效率提升了300%" | 效果对比类 |
| **反常识** | "千万别再用XXX了，这才是正确姿势" | 认知颠覆类 |
| **身份认同** | "打工人/宝妈/学生党必看！" | 精准定位类 |
| **悬念设置** | "我发现了一个AI的隐藏功能..." | 好奇驱动类 |

### 3.3 正文撰写 checklist
- [ ] 每段不超过3行，多用换行
- [ ] 关键信息用**加粗**或🔥emoji标注
- [ ] 步骤说明用「1️⃣ 2️⃣ 3️⃣」编号
- [ ] 插入至少1张效果图/对比图
- [ ] 加入个人真实使用感受（增加可信度）

### 3.4 结尾CTA（行动号召）模板
```
💬 常用话术：
• "姐妹们快试试！评论区交作业~"
• "觉得有用记得❤️⭐，下期出进阶版"
• "你们还想看什么AI工具测评？告诉我"
• "关注我不迷路，每天一个搞钱小技巧"
```

### 4.2 标签组合策略
```
标签金字塔模型：

🏔️ 顶部（1个）- 超级话题
   #AI #人工智能 #ChatGPT

⛰️ 腰部（2-3个）- 垂直领域
   #AI工具 #效率神器 #办公技巧

🌄 底部（3-5个）- 细分场景
   #打工人必备 #副业赚钱 #学生党福利 #职场干货

📍 地域（可选）
   #上海 #北京 #深圳
```

### 4.3 标签使用原则
1. **总量控制**: 每篇笔记5-8个标签为宜
2. **热度平衡**: 大热标签(100万+)搭配中小标签(1-10万)
3. **精准匹配**: 标签必须与内容强相关，避免滥用
4. **定期更新**: 关注每周热门标签变化，及时调整
