## 📐 基础规范

### 尺寸比例
| 类型 | 尺寸 | 比例 | 文件大小 |
|------|------|------|---------|
| 封面图 | 900x1200px | 3:4 | ≤500KB |
| 内页图 | 900x1200px | 3:4 | ≤500KB |
| 方形图 | 1080x1080px | 1:1 | ≤500KB |

### 配色方案
```
主色：#6366F1 (蓝紫色 - 科技感)
辅色：#8B5CF6 (紫罗兰 - 渐变)
强调：#F59E0B (橙色 - CTA)
文字：#FFFFFF (白色 - 正文)
文字次要：#E0E7FF (浅紫 - 辅助)
背景：#1E1B4B (深蓝 - 可选)
```

### 字体规范
```
标题：PingFang SC Bold / 48-60px
副标题：PingFang SC Medium / 32-40px
正文：PingFang SC Regular / 24-28px
备注：PingFang SC Light / 20-24px
```

## 🎨 封面模板库

### 模板 1: 极简大字报
**适用场景**: 产品发布、重大更新

```
┌─────────────────────────┐
│                         │
│    [emoji] 主标题      │
│                         │
│       副标题           │
│                         │
│  ──────────────────    │
│                         │
│  • 亮点 1               │
│  • 亮点 2               │
│  • 亮点 3               │
│                         │
└─────────────────────────┘
```

**设计要点**:
- 背景：纯色渐变 (#6366F1 → #8B5CF6)
- 文字：白色，居中对齐
- emoji: 1-2 个，与主题相关
- 留白：顶部 30%，底部 20%

### 模板 2: 功能清单
**适用场景**: 工具推荐、功能介绍

```
┌─────────────────────────┐
│  [Logo]  产品名        │
│  ───────────────────    │
│                         │
│  ✅ 功能 1 + 简短说明   │
│  ✅ 功能 2 + 简短说明   │
│  ✅ 功能 3 + 简短说明   │
│  ✅ 功能 4 + 简短说明   │
│  ✅ 功能 5 + 简短说明   │
│                         │
│  🎯 适用人群：xxx      │
└─────────────────────────┘
```

**设计要点**:
- 背景：浅色渐变
- 图标：统一风格 emoji
- 对齐：左对齐，行间距 1.5

### 模板 3: 对比展示
**适用场景**: Before/After 效果对比

```
┌─────────────────────────┐
│      主题标题           │
│  ───────────────────    │
│                         │
│  ┌─────┐    ┌─────┐    │
│  │Before│    │After │   │
│  │ ❌   │    │ ✅   │   │
│  │问题 1 │    │解决 1 │   │
│  │问题 2 │    │解决 2 │   │
│  └─────┘    └─────┘    │
│                         │
│  💡 核心方法：xxx      │
└─────────────────────────┘
```

**设计要点**:
- 左右对比，清晰分隔
- Before 用红色调，After 用绿色调
- 突出核心方法

## 🖼️ 内页配图规范

### 内页 1: 痛点引入
**目的**: 引发共鸣，建立需求

```
内容结构:
1. 痛点场景描述（1-2 句）
2. 常见问题列举（3-4 个）
3. 情绪渲染（emoji+ 反问）

视觉元素:
- 背景：深色渐变
- 图标：❌ ⚠️ 😫
- 文字：白色 + 黄色强调
```

### 内页 2: 解决方案
**目的**: 展示核心方法/工具

```
内容结构:
1. 方案名称（大标题）
2. 核心功能（3-5 点）
3. 使用场景（1-2 句）

视觉元素:
- 背景：品牌色渐变
- 图标：✅ ⭐ 💡
- 文字：白色 + 浅紫
```

### 内页 3: 效果对比
**目的**: 用数据/案例证明效果

```
内容结构:
1. Before 数据
2. After 数据
3. 提升百分比（突出显示）

视觉元素:
- 对比图表
- 箭头↑↓
- 高亮数字
```

### 内页 4: 使用步骤
**目的**: 降低使用门槛

```
内容结构:
1. Step 1: xxx
2. Step 2: xxx
3. Step 3: xxx
4. 注意事项

视觉元素:
- 序号：❶❷❸
- 流程图
- 截图/示意图
```

### 内页 5: 资源清单
**目的**: 提供额外价值

```
内容结构:
1. 资料包清单（3-5 项）
2. 获取方式
3. 适用人群

视觉元素:
- 📦 📁 📊 图标
- 清单列表
- 二维码/指引
```

### 内页 6: 互动引导
**目的**: 提升评论率

```
内容结构:
1. 互动问题（1 个）
2. 行动号召（评论/收藏/关注）
3. 福利提示（可选）

视觉元素:
- 💬 👇 ❤️ 图标
- 大字号问题
- 箭头指引
```

  ~/.openclaw/extensions/feishu-openclaw-plugin/skills/canghe-xhs-images/

## 📊 质量检查清单

### 发布前检查
- [ ] 尺寸正确（900x1200px 或 1080x1080px）
- [ ] 文件大小≤500KB
- [ ] 文字清晰可读（对比度高）
- [ ] 无错别字
- [ ] 品牌色一致
- [ ] 风格统一（所有图片）
- [ ] 图片数量 5-7 张

### 视觉一致性
- [ ] 封面与内页风格统一
- [ ] 配色符合品牌规范
- [ ] 字体统一（PingFang 系列）
- [ ] emoji 风格一致
- [ ] 留白比例合理
