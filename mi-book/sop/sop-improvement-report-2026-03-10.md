## 🔍 发现的问题（按优先级排序）

### P0 高优先级（必须修复）

#### 1. 视觉设计工具简陋
**问题描述**: 使用 Python PIL 生成封面图，功能有限，视觉效果一般
**影响**: 降低笔记点击率，影响品牌形象
**解决方案**:
```bash
# 1. 安装 canghe-xhs-images 技能
git clone https://github.com/sheldon-empire/canghe-xhs-images.git \
  ~/.openclaw/extensions/feishu-openclaw-plugin/skills/canghe-xhs-images/

# 2. 配置 Google Imagen API
export GOOGLE_GENMEDIA_API_KEY="your-api-key"

# 3. 使用技能生成封面
/canghe-xhs-images content.md --style minimal --layout list
```
**预计效果**: 封面质量提升 70%，点击率提升 30%
**负责人**: mi-book
**截止时间**: 2026-03-12

#### 3. 配图不完整
**问题描述**: 仅使用封面图发布，缺少内页详解图
**影响**: 降低完读率，影响收藏率
**解决方案**:
```markdown
## 配图标准（正式发布）
- 封面图：1 张（总览/吸引点击）
- 内页图：3-5 张（功能详解/步骤说明）
- 结尾图：1 张（互动引导/关注提示）
- 总计：5-7 张

## 内页图模板
1. 功能对比图（Before/After）
2. 使用步骤图（Step 1-2-3）
3. 效果展示图（数据/案例）
4. 常见问题图（Q&A）
5. 资源清单图（资料包）
```
**预计效果**: 完读率提升 40%，收藏率提升 50%
**负责人**: mi-book + mi-hua
**截止时间**: 2026-03-12

### P1 中优先级（建议优化）

#### 4. 选题库未建立
**问题描述**: 每次临时搜索爆款，效率低且质量不稳定
**影响**: 选题质量波动大，无法持续产出爆款
**解决方案**:
```markdown
## 选题库结构
- 爆款标题模板（10 个）
- 选题池（20 个储备选题）
- 竞品监控表（5 个对标账号）
- 热点日历（节日/事件/趋势）

## 选题评分机制
| 维度 | 权重 | 评分标准 |
|------|------|---------|
| 热度 | 40% | 搜索量/互动量 |
| 匹配度 | 30% | 与账号定位契合度 |
| 可行性 | 20% | 内容创作难度 |
| 差异化 | 10% | 与竞品区别度 |
```
**预计效果**: 选题效率提升 60%，爆款率提升 25%
**负责人**: mi-book + mi-zhi
**截止时间**: 2026-03-17

#### 5. 无发布前预览
**问题描述**: 无法在发布前确认最终效果
**影响**: 发布后发现问题无法撤回（小红书不支持编辑）
**解决方案**:
```bash
# 理想流程
1. 调用 /api/v1/preview 生成预览链接
2. 在浏览器中打开预览
3. 确认无误后调用 /api/v1/publish
4. 如有问题，修改后重新预览
```
**预计效果**: 发布错误率降低 90%
**负责人**: mi-book
**截止时间**: 2026-03-20
