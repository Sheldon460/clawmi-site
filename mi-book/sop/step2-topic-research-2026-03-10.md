## 2.2 爆款规律分析

### 藏赞比解读
- **藏赞比 > 2.0**: 超高收藏价值（工具汇总、教程类）
- **藏赞比 1.0-2.0**: 正常收藏价值（技能分享、经验类）
- **藏赞比 < 1.0**: 情绪价值为主（观点、故事类）

### 爆款标题公式

#### 公式 1: 数字 + 效果 + 情绪词
```
「41 个 Openclaw 应用场景汇总」→ 数字具体 + 场景明确
「11 种利用 AI 提升效率的方法」→ 数字具体 + 效果清晰
```

#### 公式 2: 身份标签 + 痛点 + 解决方案
```
「gap 两年多已找到工作，分享一个面试顶级心态」→ 身份共鸣 + 痛点解决
「空窗期两年，我凭这段话拿下 5 个 Offer」→ 身份共鸣 + 结果导向
```

#### 公式 3: 产品/工具 + 动作 + 效果
```
「飞书技能包上线🎉」→ 产品 + 动作 + 情绪
「一条命令，让 openclaw 指挥一支 7 人团队」→ 低门槛 + 高效果
```

### 高互动内容特征

1. **数字具体**: 41 个、11 种、7 人团队、5 个 Offer
2. **效果可量化**: 效率提升、拿下 Offer、指挥团队
3. **情绪共鸣**: gap 空窗期、打工人、面试焦虑
4. **低门槛高回报**: 「一条命令」「一句话」「一个方法」
