## ⚡ 3步完成发布

### Step 1: 生成封面图（幂画）
**预计时间**: 3-5分钟

1. 打开即梦AI：https://jimeng.jianying.com
2. 点击「图片生成」
3. 粘贴提示词：
   ```
   小红书风格知识分享封面图，主题"2025必看5个免费AI工具"。渐变紫蓝色背景，中央大号白色粗体文字"5个免费AI工具"，顶部红色火焰emoji和"2025必看"字样，底部白色文字"效率提升300%"带向上箭头图标。装饰元素：右上角有白色机器人头像轮廓，左下角有闪电图标，整体氛围专业、现代、吸睛，高清质感，3:4竖版比例。
   ```
4. 设置比例：3:4竖版
5. 点击生成，等待结果
6. 下载高清封面图，保存到本地

**备用方案**：
- 可画Canva: https://www.canva.cn
- 稿定设计: https://www.gaoding.com
- 使用 `cover-design.md` 中的设计规范

### Step 2: 发布到小红书（幂书）
**预计时间**: 3-5分钟

1. 登录小红书创作服务平台：
   https://creator.xiaohongshu.com

2. 点击「发布笔记」→ 选择「图文」

3. 上传图片：
   - 第1张：封面图（刚刚生成的）
   - 第2-6张：工具介绍图（可用文字截图或占位图）
   - 第7-9张：总结页、引导关注页

4. 填写标题：
   ```
   🔥 2025职场人必看｜5个免费AI神器让我效率翻倍！
   ```

5. 填写正文（复制以下内容）：
   ```
   姐妹们！今天分享让我工作效率提升300%的宝藏工具💎
   全部免费！学生党职场新人都能用！建议收藏⭐

   ❶ ChatGPT / Claude
   ✅ 写文案、改简历、做PPT大纲
   ✅ 日常问题解答，24小时在线
   💡 提示：用英文提问回答更专业！

   ❷ 讯飞听见
   ✅ 会议录音实时转文字
   ✅ 1小时录音5分钟出稿
   ✅ 支持中英双语识别
   💡 提示：关键会议一定要录音备份！

   ❸ 美图AI / 即梦AI
   ✅ 小红书封面一键生成
   ✅ 商品图背景自动替换
   ✅ 模特换装不用重拍
   💡 提示：输入越详细，出图越精准！

   ❹ Notion AI
   ✅ 会议纪要自动整理
   ✅ 长文档一键摘要
   ✅ 头脑风暴灵感生成
   💡 提示：配合模板使用效率翻倍！

   ❺ Gamma / 秒出PPT
   ✅ 输入大纲自动生成PPT
   ✅ 精美模板免费使用
   ✅ 支持在线协作编辑
   💡 提示：先写内容大纲再生成更省时！

   📌 使用心得：
   工具在精不在多！先掌握1-2个用到极致
   比下载10个积灰强100倍💪

   你们平时用什么AI工具？评论区聊聊👇
   ```

6. 添加标签（选择9个）：
   - #AI工具
   - #效率神器
   - #职场干货
   - #打工人必备
   - #自我提升
   - #干货分享
   - #效率提升
   - #小红书学习
   - #宝藏工具

7. 设置发布时间：
   - 选择「立即发布」或设定「12:00」定时发布

8. 点击「发布笔记」
