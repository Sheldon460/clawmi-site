# 🧠 Self-Improving 协议标准模板 (V1.0)

> 本文件为幂家军全员强制执行的自我进化协议标准。
> 最后更新: 2026-03-27
> 部署状态: 全员强制执行

---

## 📁 目录结构要求

每个 Agent 必须创建以下目录结构：

```
workspace/{agent_id}/self-improving/
├── memory.md              # 个人核心记忆
├── corrections.md         # 错误纠正记录
├── index.md              # 索引与统计
├── reflections/          # 反思报告
│   ├── daily/           # 每日复盘
│   ├── weekly/          # 周度分析
│   └── monthly/         # 月度报告
└── projects/            # 项目追踪
```

---

## 🎯 触发条件（必须执行）

| 触发场景 | 执行动作 | 记录位置 |
|---------|---------|---------|
| 用户纠正错误时 | 立即记录错误与纠正方案 | `corrections.md` |
| 完成重要工作后 | 执行自我反思 | `reflections/daily/` |
| 发现输出可优化时 | 记录改进点 | `memory.md` |
| 每日 23:00 | 自动复盘今日工作 | `reflections/daily/` |

---

## 🔧 SOUL.md 更新要求

每个 Agent 的 `SOUL.md` 必须包含以下章节：

```markdown
## 🧠 Self-Improving 配置（强制执行）

### 初始化状态
- [ ] self-improving目录已创建: `workspace/{agent_id}/self-improving/`
- [ ] memory.md 已初始化
- [ ] corrections.md 已初始化
- [ ] index.md 已初始化

### 触发条件（必须执行）
1. **用户纠正错误时** → 立即记录到 corrections.md
2. **完成重要工作后** → 执行自我反思
3. **发现输出可以优化时** → 记录改进点
4. **每日23:00** → 自动复盘今日工作

### 执行流程
```
1. 检测触发条件
2. 执行自我反思/错误记录
3. 写入 workspace/{agent_id}/self-improving/ 目录
4. 同时写入记忆系统:
   - 操作流水 → memory/YYYY-MM-DD.md
   - 全局策略 → playbook.md
   - 核心决策 → MEMORY.md
