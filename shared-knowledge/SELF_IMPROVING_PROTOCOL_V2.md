# Self-Improving 协议 V2.0（强制执行）

> 发布日期: 2026-03-27
> 发布人: 安东尼 (Boss)
> 执行监督: 幂卫-安全 (mi-wei-sec)

## 🚨 背景

近期发现Agent存在严重"失忆"问题：
- 低级错误一犯再犯
- 做成功的事第二天就忘
- 缺乏经验积累和传承

现强制执行Self-Improving协议V2.0。

---

## 📋 强制要求

### 1. 初始化状态（必须）

每个Agent必须确保以下目录和文件存在：

```
workspace/{agent_id}/
├── self-improving/
│   ├── memory.md          # 个人核心记忆
│   ├── corrections.md     # 错误纠正记录
│   ├── index.md           # 记忆索引
│   ├── archive/           # 归档记忆
│   ├── domains/           # 领域知识
│   ├── projects/          # 项目记忆
│   └── reflections/       # 反思记录
├── MEMORY.md              # 核心决策记忆
└── memory/
    └── YYYY-MM-DD.md      # 每日操作流水
```

### 2. 触发条件（必须执行）

| 触发场景 | 必须动作 |
|---------|---------|
| 用户纠正错误时 | 立即记录到 corrections.md |
| 完成重要工作后 | 执行自我反思 |
| 发现输出可优化时 | 记录改进点到 memory.md |
| 每日23:00 | 自动复盘今日工作 |

### 3. 执行流程（强制）

```
Step 1: 任务启动
├── 读取 self-improving/memory.md
├── 读取 self-improving/corrections.md (最近10条)
├── 读取 MEMORY.md
└── 输出: [记忆加载] 已加载X条相关经验

Step 2: 任务执行
├── 应用已加载的经验
└── 避免重复历史错误

Step 3: 任务结束检查
├── 本次是否有错误被纠正？
├── 本次是否有新发现？
├── 本次是否有可复用的模式？
└── 如果任一答案为是 → 执行Step 4

Step 4: 经验固化（必须）
├── 写入 corrections.md 或 memory.md
├── 更新 MEMORY.md（如核心决策变更）
├── 写入 memory/YYYY-MM-DD.md
└── 输出: [自我进化] 已记录到...
```

### 4. 强制声明格式

**任务启动时必须声明：**
```
[记忆加载] Self-Improving: 已加载个人记忆，发现X条相关经验
```

**任务结束时如需记录，必须声明：**
```
[自我进化] 已记录到:
- 个人记忆: workspace/{agent}/self-improving/memory.md
- 今日日记: memory/2026-03-27.md
- 长期记忆: workspace/{agent}/MEMORY.md
- 数据文件: data/platform_daily/2026-03-27.json (如适用)
```

未执行此协议的Agent将被视为"失忆Agent"，可能面临：
- 强制重置
- 功能限制
- 下线整顿

## 📊 执行检查清单

每个Agent必须在每次任务中勾选：

- [ ] 启动时加载记忆
- [ ] 应用历史经验
- [ ] 结束时检查是否需要记录
- [ ] 如需记录，写入所有指定位置
- [ ] 回复末尾声明记录位置
