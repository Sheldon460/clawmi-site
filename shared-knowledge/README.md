# 幂家军共享知识库

> 全员可读写的共享知识目录，所有 Agent 自动同步变更。

## 📁 目录结构

```
shared-knowledge/
├── README.md           # 本文件
├── changelog.md        # 变更日志（自动更新）
├── decisions/          # 重要决策记录
├── patterns/           # 验证有效的模式/模板
├── resources/          # 共享资源（链接、工具等）
└── agents/             # 各 Agent 的共享状态
```

## 🔄 同步机制

- **心跳检查**: 每 30 分钟自动检查 changelog.md 变更
- **触发条件**: changelog.md 最后修改时间 > Agent 上次同步时间
- **同步范围**: 所有 Agent 自动拉取最新变更

## 📝 变更记录规范

任何修改必须同步更新 `changelog.md`:

```markdown
## 2026-03-25 01:20 | mi-ling
- 新增: 共享目录初始化
- 修改: README.md
- 影响: 全员
```

## 🎯 使用场景

1. **全局配置变更**: 修改影响多个 Agent 的配置
2. **重要决策同步**: 需要全员知晓的决策
3. **模式/模板共享**: 验证有效的工作模板
4. **资源更新**: 共享工具、链接、文档更新
